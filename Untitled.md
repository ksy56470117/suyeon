상행위란?::영리 목적으로 행해지는 법률행위
주관주의란?::상인을 먼저 정의하고 상행위 개념을 정하는 방식
영업적 상행위 vs 보조적 상행위::영업적=상인이 영업으로 하는 행위 / 보조적=상인이 영업을 위하여 하는 행위
"상인이 영업을 위하여 하는 행위"의 효과::상행위로 간주
"상인의 행위"의 효과::영업을 위한 것으로 추정 (간주 아님)
일방적 상행위란?::1인에게만 상행위여도 전원에 상법 적용
쌍방적 상행위란?::둘 다 상인일 때만 적용 (예: 일반상사유치권)
상사소멸시효 기간::5년 (민사 10년)
상사법정이율::연 6%
상사유치권 성립요건 5가지::당(쌍방 상인)·피(쌍방적 상행위)·목(채무자 소유+채권자 상행위로 점유)·관(견련성 불필요)·반(반대특약 없을 것)
상사질권에서 유질계약::유효 (민법은 무효) / 질권설정자가 상인 아니어도 유효
상사채무 연대책임 적용 기준::채무자에게 상행위가 될 때만 (채권자 무관)
상사보증인 연대책임 성립 조건::보증 자체가 상행위이거나 주채무가 상행위
상사보증과 민법보증의 차이::상법=최고·검색의 항변 불가 / 민법=가능
보수청구권 요건::상인이 ①영업범위 내에서 ②타인을 위해 행위 시 상당한 보수 청구 가능
비현명주의 예외::어음행위는 엄격한 현명주의 적용
낙부통지의무 해태 시 효과::승낙 간주
무상임치 주의의무::영업범위 내 무상임치도 선관주의 의무
상사매매 요건 3가지::①쌍방 상인 ②쌍방적 상행위 매매 ③반대특약 없을 것
상사매매 공탁·경매권 — 법원허가 필요한가?::불필요 (민법은 필수)
확정기매매 해제 (상법) — 해제의사표시 필요한가?::불필요 (기간 경과만으로 자동 해제)
매수인 검사·통지의무 (§69) 성질::간접의무 (위반해도 강제이행·손배 청구 불가, 단 해제·감액 청구 불가)
매수인 검사·통지의무 즉시발견불가 시 기한::6개월 내 통지
매수인 보관·공탁·경매의무 (§70,71) 성질::진정한 의무 (위반 시 손배 책임)
상호계산 불가분의 원칙 효과::독립성 상실 — 채권행사·임의제거·시효진행 모두 불가
상호계산 시효 기산점::잔액 확정 시부터
상호계산 해지 시기::언제든지 (기간 정해도 예고 없이 가능)
익명조합 이익분배::필수 / 손실분담은 필수 아님
익명조합 출자 가능 종류::금전·현물만 가능 (노무·신용 불가)
익명조합 출자재산 귀속::영업자 단독소유
익명조합 종료 사유 (영업자)::사망·금치산·파산
익명조합원 vs 영업자 — 지위 양도/상속::익명조합원=양도✕ 상속○ / 영업자=양도✕ 상속✕
상사소멸시효 적용 ○ 사례::채무불이행 손배청구권, 계약해제 원상회복청구권, 급부 자체의 부당이득반환
상사소멸시효 적용 ✕ 사례::불법행위 손배청구권, 상행위 무관 부당이득반환
