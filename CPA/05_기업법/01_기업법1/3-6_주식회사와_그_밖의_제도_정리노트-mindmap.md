# 3-6. 주식회사와 그 밖의 제도 — CPA 체계 정리노트


> 관련: [3-5 이사회](./3-5_2절_이사회와_대표이사_정리노트.md) · [3-4 주주총회](./3-4_1절_주주총회_정리노트.md) · [3-3 주식개관](./3-3_1절_주식과_주주의_개관_정리노트.md)


# Part A. 자본금의 증가 — 신주발행


## A1. 절차

- 수권자본제도: 이사회 결정 (정관→주총도 가능)
- 배정: 원칙 **주주배정**(주주평등) / 예외: 법률·정관 → 제3자

## A2. 신주인수권

- 추상적(주식분리양도 ✕) → 구체적(분리양도 ○) / 증서: 양도수단+청약수단
- 증서: 선의취득 ○ / 제권판결 ✕(단기) / 추정력 ○

## A3. 납입

- **현실납입**(면제 ✕·대물변제 ✕) / **전액납입**(분할 ✕)
- 출자전환(상계): 설립시 절차없이 ○ / 신주발행시 **회사동의** 필요
- 현물출자: 정관기재(신주발행시 불필요)·검사인조사·청약서 기재 / 면제: 1/5+5천만·거래소시세·대통령령·**출자전환**

## A4. 효력발생·등기

- 납입기일 **다음날** 주주권 발생 / 납입해태 → **당연실권**
- 변경등기: 본점2주 / **창설적효력 ✕**

## A5. 신주발행 유지청구 vs 위법행위유지청구 ★★★★

| | 위법행위유지(§402) | 신주발행유지(§424) |
|---|---------|---------|
| 취지 | 회사 손해방지 | **주주** 불이익방지 |
| 청구자 | 감사 ○ / 1/100주주 | 감사 ✕ / **단독주주권** |
| 상대방 | 이사 | **회사** |
| 사유 | 법령·정관위반 | **불공정도 가능** |
| 공통 | 사전수단 / 소+소외 | |

## A6. 신주발행무효의 소 ★★★★

- 원고: 주주/이사/감사 / 발행일+**6월**
- 무효사유: 액면미달위반·현물과대평가·발행예정초과·**경영권방어배정**
- ✕ 무효사유: 이사회결의 흠결·검사인조사 흠결만
- 판결: 대세효+**불소급효** / 주주→주권반납 / 회사→납입금 환급


# Part B. 자본금의 감소


## B1. 거지감자(실무99%) vs 부자감자

- 거지감자(결손보전): **총회보통결의** + 채보절차 ✕ (주권제출기간만료→효력)
- 부자감자: 총회**특별**결의 + 채보절차 ○

## B2. 감자방법: 병합/강제소각/임의소각

## B3. 감자무효의 소 (§445)

- 원고: 주주/이사/감사/청산인/파산관재인/**불승인채권자** / 변경등기+6월
- 판결: **소급효** ○ (cf. 총회결의하자4종+감자무효 = 소급효)


# Part C. 정관변경

- **총회특결** / 종류총회(정.구.회) / 효력: 총회결의 있은 때 (공증인인증·변경등기 ✕)


# Part D. 이익배당


## D1. 배당가능이익

- = 순자산액 − (자본금 + 법정준비금 + 당기적립이익준비금 + 미실현이익)

## D2. 방법

- 주주평등(주식수비례) / 예외: 종류주식·대주주양보·현물배당군소주주우대
- 지급: 결의+1월 내 (따로 지정 가능) / 청구권시효: **5년**

## D3. 위법배당 ★★★★★

- 총회결의**무효**확인의 소 / 반환: 선악·과실 **불문** / 채권자 직접 반환청구(채권보전불문)
- 이사: §399+§401 / 경영판단법칙 ✕ / 2년 지나도 책임해제 ✕ (부정행위)

## D4. 주식배당 (§462의2)

- 주총결의(이사회 ✕) / 이익배당 총액 **1/2** 이내 / 권면액 기준
- 신주주: 총회 **종결** 한 때 (cf. 준비금전입: 이사회→기준일 / 주총→결의있은때)

## D5. 준비금의 자본금 전입 ★★★★

- 법정준비금(이익+자본) / 사용: 결손보전·자본전입 / 사용순서: **무관**
- 결의: 원칙 이사회 / 예외 정관→주총 / 법적성질: **주식분할설(판례)**
- 질권: 물상대위(§339) 약식·등록 **공통** 적용
