# 3-5. 감사와 감사위원회 — CPA 체계 정리노트


> 관련: [3-5 이사회](./3-5_2절_이사회와_대표이사_정리노트.md) · [3-4 주주총회](./3-4_1절_주주총회_정리노트.md)


# Part A. 감사


## A1. 지위

- 원칙: **필수기관** / 예외: 10억미만 → 임의 / 유한회사 → 임의
- 회사vs이사 소송 → **감사가 회사 대표** (해임된 이사 → 대표이사가 대표)

## A2. 선임·종임

- 주총 **보통결의** / 3/100 초과 의결권 ✕ / 전자투표 → 출석과반수만
- 임기: 취임후 **3년 내 최종결산기 정기총회 종결시** (단축 불가 / 연장 당연)
- 해임: 총회특결 / 감사는 **의견진술권** ○

## A3. 권한 (수인의 감사 → 각자 **단독** 행사)

### 주된 권한: 업무감사권 + 회계감사권 (북두칠성그림)

### 부수적 권한

- ① 보고요구/조사권 ② 위법행위유지청구 ③ 이사회소집청구 ④ 이사회출석·의견진술
- ⑤ **임시총회소집청구** ⑥ 회사대표권(§394) ⑦ 자회사 조사 ⑧ 각종 소권(원고삼총사)

### 이사회에서 감사 = observer (member ✕)

- 권리 3: 출석권·의견진술·소집통지받을권리 / 의무 1: 의사록 기명날인

## A4. 의무

- 선관주의 (set menu ✕) / 감사보고서 작성(4주내) / 이사회 보고 / 총회 의견진술 / 감사록작성

## A5. 책임

- to 회사: 임무해태(경과실도) / to 제3자: 악의·중과실 / 이사와 **연대**


# Part B. 감사위원회


## B1. 구성

- 이사회 내 위원회 / **3인↑, 2/3↑ 사외이사** / 감사와 병존 불가

## B2. 선임·해임·운영

- 일반회사: 이사회 결의 / 대규모상장: **주총결의** (3/100 의결권제한)
- 해임: 이사 총수 **2/3↑** 찬성 / 운영: 이사회 준용 / 하자: 민법 당연무효

## B3. 핵심 차이

- 결의 → 이사통지 → 이사회 번복 **불가** (독립성 보장)
- 대표위원 선정 필수 (공동대표 가능) / 회사비용 전문가 조력 가능


# Part C. 상장회사 특례 ★★★★


## C1. 모든 상장회사

- 이사·감사 선임/보수 → **별도 상정** / 감사보고서: 주총1주전까지(4주내 아닌)
- 감사 선임·해임: 3/100 초과 의결권 ✕ (최대주주: 특수관계인 합산)

## C2. 대규모상장(2조↑) — **"대규모감위 필·총·위·대"**

- **필**: 감사위원회 필수 / **총**: 주총결의 선임·해임(3/100제한)
- **위**: 최소1인 **분리선출** / **대**: 대표=사외이사, 1인↑ 회계·재무전문가

### 분리선출 ★★

- 일괄선출(보수): 이사선임→감사위원선임 / 분리선출(진보): 최소1인 분리
- 현행: **일괄선출 프로세스 + 최소1인 분리선출** (타협안)
- 일괄선출 해임 → 이사자격 유지 / **분리선출 해임 → null(이사도 상실)**

## C3. 1천억↑ 상장회사

- **상근감사 1인↑** 필수 (감사위원회 있으면 면제)

## C4. 준법지원인 (5천억↑)

- 이사회 임면 / 변호사·법학교수 등 / 임기 **3년** 상근 / 비밀유지(**감.이.집.준.대.가**)
- 다른법률 임기 < 3년 → **상법 3년 우선적용**
