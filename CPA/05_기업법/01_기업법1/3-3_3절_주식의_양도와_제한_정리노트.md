# Part A. 양도 원칙

## A1. 양도방법
- 효력요건: **양도계약 + 주권교부** (현실인도+관념적 인도 모두 가능)
- 회사 대항요건: **명의개서** / 제3자 대항요건(이중양도): **주권점유** 기준

## A2. 판례
- 대금미지급 → 양도계약 해제 → 주권 미반환이어도 **주주지위 상실**
- 주주지위 재획득 → 회사 주주권 행사에는 **명의개서 복구** 필요

# Part B. 양도의 제한

## B1. 권리주 양도금지 (§319)
- 회사에 대해 법적효력 ✕ / 양도인-양수인 간 **채권적 효력만** ○

## B2. 주권발행전 양도제한 (§335-3)
- 원칙: 양도효력 ✕ / 예외: 주식효력발생후 **6월 경과** + 주권미발행 → 양도효력 ○
- 6월 경과후: 민법 **지명채권 양도방식** + 명의개서 청구
- 이중양도: **확정일자+통지 빠른 순서**

## B3. 자기주식 취득제한 (§341)

### 취득제한 이유 4가지
- ① 자본충실 침해 ② 투기수단 ③ 회사지배 왜곡 ④ 주주평등 위반

### 예외 — 배당가능이익 취득
- 자본충실요건 + 주주평등요건(거래소/균등조건) / 결의: 재무제표 승인기관

### 예외 — 특정목적 취득 (**합.영.권.단.주**)

### 위법한 자기주식 취득 → **깡그리 무효** (채권적 효력도 ✕)

### 적법 자기주식 처리
- 의결권 행사 ✕ / 처분(이사회) / 소각(감자·이익) / 계속보유 가능

## B4. 자회사의 모회사주식 취득제한 (§342-2)
- 원칙: 취득 ✕ / 예외: 합영 + 포괄적교환이전 + 권리실행 + 삼각set → 6월내 처분
- 위반 → **깡그리 무효**

## B5. 정관상 양도제한 (§335-1)
- 승인기관: **이사회에 한정** (cf. 유한회사: 아무 제한이든 가능)
- 승인없는 양도: **채권적 효력만** ○ / 비상장회사만 적용
- 절차: **승/거/지/통/매** (승인청구→거부→상대방지정or주식매수→통지→매도)

## B6. 강제매수제도

### 지배주주(95/100↑) 매도청구
- 주총사전승인 + 1월전 공고통지 + 2월내 매도 / 이전시점: 매매가액 지급 or 공탁

### 소수주주 매수청구
- 절차 간소(사전승인·공고통지 불필요) / 매수청구일+2월내
