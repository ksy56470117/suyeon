# Part B. 대리상 (§1절)

## B1. 개념
- 일정한 상인을 위하여 / 상업사용인이 아니면서 / 상시 영업부류 거래의 대리·중개를 영업
- 체약대리상(법률행위) + 중개대리상(사실행위=중개) → 둘 다 당연상인(기본적 상행위)

## B2. 의무
- ① **통지의무(§88)**: 대리·중개한 때 지체없이 본인에게 통지
- ② **경업금지(§89)**: 본인 허락 없이 거래금지+겸직금지 / 위반 → 개·손·해
- ③ **영업비밀준수(§92-3)**: 계약종료 후에도

## B3. 권리
- 보수청구권(§61) / 통지받을 권한(§90) / **특별상사유치권(§92)** / 보상청구권(§92-2)

### 특별상사유치권 — "회장아들차사건"

| 요건 | 내용 |
|------|------|
| 당 | 둘 다 상인 |
| 피 | 쌍방적 상행위 채권 |
| 목 | 채무자 소유 한정 ✕ / 상행위로 점유 불필요 |
| 관 | 견련성 불필요 |
| 반 | 반대특약 없을 것 |

### 보상청구권 (§92-2)
- 본인 새 고객 획득/거래 현저 증가 → 계약종료 후에도 이익 → 보상 청구 가능
- 한도: 종료 전 **5년간 평균연보수액** / 소멸: 종료일+**6월**

### 해지 (§92)
- 기간미정 → 2월 전 예고 / 부득이 → 언제든지

# Part C. 중개업 (§2절)

## C1. 개념
- 타인간의 **상행위의 중개**를 영업 → 불특정상인 보조 → 경업금지 ✕
- 민사중개인(공인중개사·결혼정보회사)도 상인이지만 상법상 중개인은 아님

## C2. 의무
- ① 견품보관(§95): 행위 완료까지 보관
- ② 결약서교부(§96): 지체없이 / 중개인 기명날인 필수
- ③ 장부작성(§97): 일기장 / 당사자 언제든지 등본 교부 청구
- ④ 성명/상호 묵비(§98): 당사자 요구 시 결약서·장부등본에 미기재
- ⑤ **이행담보(개입의무)(§99)**: 묵비한 경우 상대방이 중개인에게 이행 청구

## C3. 권리
- 보수청구권: **결약서 교부 종료 후** (이행 시점 ✕)
- 보수: 당사자 쌍방 **균분** 부담
- **Not 권리**: 급여수령대리권 ✕ / 특별상사유치권 ✕

---

# Part D. 위탁매매업 (§3절)

## D1. 개념 (§101)
- **자기의 명의 + 타인의 계산**으로 물건/유가증권의 매매를 영업
- 위탁매매인은 제3자에 대해 직접 권리취득·의무부담 (§102)

## D2. 의무
- ① 통지/계산서 제출(§104) / ② **이행담보=개입의무(§105)**: 상대방 불이행 → 위탁자에 이행 책임
- ③ 지정가액 준수(§106) / ④ 하자통지·처분(§108)

## D3. 권리
- 보수청구권 / **전면적 개입권(§107)**: 거래소 시세 있는 물건 → "내가살게/팔게"
- **특별상사유치권(§111)**: 대리상 준용 (당·피는 차이有)

### 특별상사유치권 당·피 차이 (대리상 vs 위탁매매인)

| 요건 | 대리상 | 위탁매매인 |
|------|--------|----------|
| 당 | 둘 다 상인 | 채무자 상인 **불필요** |
| 피 | 쌍방적 상행위 | **일방적도 가능** |

### 위탁물의 귀속 (§103)
- 위탁매매인의 채권자 간 관계에서 **위탁자의 소유/채권으로 봄** → 위탁자 보호

# Part E. 운송업 (§4절) → [참조: 운송주선업](#part-f-운송주선업)

## E1. 물건운송인의 의무·책임

### 손해배상(§135) — 과실책임주의
- 증명책임: **운송인** 부담

### 손배책임 제한 (**정·고·특·단**)

| 제한 | 내용 |
|------|------|
| **정**액배상(§137) | 도착지 가격 기준 / 고의·중과실 → 모든 손해 |
| **고**가물특칙(§136) | 종류·가액 명시 시만 책임 / 미명시 → 보통물값도 ✕ |
| **특**별소멸(§146) | 유보없이 수령+운임지급 → 책임소멸 (즉시발견불가 → 2주내 통지) |
| **단**기시효(§147) | 수령일+1년 / 전부멸실=인도할 날+1년 / 악의 → 5년 |

### 화물상환증(화증)
- 법적성질: **상환증권 + 당연한 지시증권 + 처분증권**
- 채권적효력: 화증소지인 → 운송인에게 인도청구권
- **물권적효력(§133)**: 화증교부 = 운송물 인도와 동일 → 소유권 이전

### 순차운송(§138)
- 각 운송인 연대배상 / 구상권 ○ / 원인불명 → 운임액 비율 분담

## E2. 여객운송
- 인적손해(§148): 과실책임 / 비정액적·비정형적
- 탁송수하물(§149): 무료라도 물건운송인 동일 책임 / 도착일+10일 미인도 → 공탁/경매
- 휴대수하물(§150): **여객이 증명책임**

# Part F. 운송주선업 (§5절)

## F1. 개념 (§114)
- **자기의 명의 + 위탁자의 계산**으로 물건운송의 주선을 영업 → 위탁매매인 구조와 동일

## F2. 운송인과의 차이

| 구분 | 운송인 | 운송주선인 |
|------|--------|----------|
| 책임원인 | 운송주선인 잘못도 책임 | 운송인 선택 주의해태만 |
| 손배제한 | 정/고/특/단 | **고/단** (정액배상 ✕) |

## F3. 전면적 개입권 (§116)
- 거래소 시세 **불필요** (cf. 위탁매매인: 필수) / 직접 운송 시 운송인 동일 권리의무

---

# Part G. 공중접객업·창고업·새로운 상행위

## G1. 공중접객업 (§152)
- 임치받은 물건: 과실책임 → **업자 증명책임** / 휴대물: **고객 증명책임**
- 정고특단 중 **고·단**만 / 시효 **6개월** (전부멸실 → 퇴거일부터)

## G2. 창고업 (§160)
- 과실책임 / 정고특단 중 **특·단**만 / 시효 1년 (전부멸실 → 멸실통지 발송일+1년)
- 창고증권 = 화물상환증 준용

## G3. 새로운 상행위
- **금융리스**: 이용자 선정 재산 → 리스업자 취득·대여 → 이용자 이용
- **가맹업**: 가맹업자 의무(지원·영업지역 보호) / 가맹상 의무(권리침해방지·비밀준수)
- **채권매입업(팩토링)**: 영업채권 매입·회수 / 상환청구 가능(다른 약정 없는 한)
