# Part A. 회사의 개념

---

## A1. 회사의 의의 (§169)

> **정의: 상행위(22종) or 그 밖의 영리를 목적으로 설립한 법인**

- 상사회사 (상행위 22종) / 민사회사
- 2011년 개정: "사단" 요건 삭제

### A1-1. 영리성
- (가) 이윤추구활동 후 이익을 **사원에게 분배**하는 것
- (나) 사원 = 지분을 갖고 있는 자

### A1-2. 사단 ← 2011년 삭제
- (가) 사단 = 단체성이 강한 모임 (cf. 조합 = 단체성이 약한 모임)
- (나) 물적회사는 1인 회사 가능 → 사단 요건 불필요해짐
- (다) 인적회사의 내부관계 → 민법 조합규정 적용

### A1-3. 법인
- (가) 사원과 회사 = 별개 인격 = **책임의 분리**
- (나) 성립시기: **본점소재지에서 설립등기한 때**
- (다) 법인 분류
  - ① 영리법인 → 회사
  - ② 비영리법인
    - ⓐ 공익법인: 재단법인(학교·의료법인) / 비영리사단법인
    - ⓑ 일반공법인 (예: 서울특별시)

> ★ 정관 = 법원(法源) ○ / 약관 = 법원 ✕

## A2. 1인회사

### A2-1. 유형별 인정여부

| 구분           |    1인 설립     |     1인 존속     |
| ------------ | :----------: | :-----------: |
| 인적회사 (합명·합자) | ✕ (2인↑ 정관작성) | ✕ (1인 남으면 해산) |
| 주식·유한·LLC    |      ○       |       ○       |

### A2-2. LLC 특징
- (가) **내부**관계 → 합명회사(인적) = 민법 조합 준용
- (나) **외부**관계 → 물적회사 (사원 아닌 자도 업무집행자 가능)

### A2-3. 1인회사 — 주주총회 하자 ★★★★★

#### (참고) 주총 소집절차 4단계
- ① 이사회 결의로 총회소집
- ② 대표이사가 소집집행
- ③ 주주에게 소집통지 (2주 전, 서면/전자문서)
- ④ 통지 시 목적사항 기재

#### 하자 유형별 소의 종류

| # | 하자 내용 | 소의 종류 |
|---|----------|----------|
| ① | 이사회 결의 없이 총회소집 (경미) | 취소의 소 |
| ② | 대표이사 소집집행 없이 진행 | **부존재확인의 소** |
| ③-a | 소집통지 일부 누락 | 취소의 소 |
| ③-b | 소집통지 대부분 누락 | **부존재확인의 소** |
| ④ | 목적사항 미기재 | 취소의 소 |

#### 1인회사 결론
- ①②③④ 어느 것을 위반해도 **1인주주의 의사에 합치** (= 1인주주 뜻대로 의사록 작성) → **총회결의 유효**
- 복수 주주를 전제로 한 규정 → 1인회사에 적용 ✕

### A2-4. 1인회사 — 자기거래
- 1인주주 → **이사회 승인 불요**

### A2-5. 1인회사 — 기타
- (가) 1인회사라도 회사와 주주 = **별개의 인격**
- (나) 1인주주가 회사 재산 감소 → **횡령죄/배임죄 성립**
- (다) 매매 자체는 유효하게 인정, 형사책임은 별도

| 관계 | 효력 |
|------|:----:|

- (나) **정관에 규정 있음** → 이사회승인 부재를 이유로 회사가 책임회피 불가

# Part B. 법인격부인론

---

## B1. 개념
- (가) 법인격 남용 시 → 법인격 **박탈 ✕** / 회사와 사원의 인격을 **동일시**
- (나) 목적: **회사채권자 보호**
- (다) 효과: 회사채무를 **지배주주에게 전가**
- (라) ★★ 회사와 주주의 독립성이 사실상 사라짐 (법인격 박탈은 아님)

## B2. 적용요건 (판례 — 민법 §2 신의칙)
- ① 지배주주의 **개인사업체**처럼 운영
- ② 지배주주에 대한 **법률적용 회피 수단**으로 운영

## B3. 적용효과 3가지
- ① 회사의 법인격은 **박탈되지 않음**
- ② 회사 채무 → 지배주주가 변제 책임 (회사 책임도 소멸 ✕)
- ③ 회사에 대한 판결의 효력이 지배주주에게 **미치지 않음**
  - → 회사 승소판결만으로 지배주주에게 강제집행 불가
  - → **회사 + 지배주주를 공동피고로 제소** → 각자에 대한 집행권원 취득 필요

## B4. 역적용 — 사해설립 (Reverse Piercing)

### B4-1. 개념
- 일반: 회사채무 → 지배주주에게 전가
- **역적용**: 설립자의 채무 → **회사에 전가**

### B4-2. 일반적 구제수단

| # | 수단 | 가부 | 이유 |
|---|------|:----:|------|
| ① | 지배주주 보유 주식에 강제집행 | ○ | — |
| ② | §176 회사해산명령 | ✕ | 공익적 규정 → 사적채권자 이용 불가 |
| ③ | §185 사해설립취소의 소 | 사실상 ✕ | 합명·합자·유한·LLC만 가능, **주식회사엔 없음** |

→ 사기꾼들이 주식회사를 선호하는 이유

### B4-3. 역적용 인정 ④ — 판례 (안건사 사건)
- 기존회사(주안건사) 재산 → 신설회사(주안건미디어)로 이전
- 대법원: 채권자는 기존·신설 **어느 쪽에도 이행 청구 가능**

# Part C. 회사의 종류 — 6종 상세

---

## C1. 합명회사

### C1-1. 사원의 책임

> **무한 / 직접 / 연대책임** (순서·비율 불문)

#### (1) 책임발생 요건 (§212)
- ① 회사재산으로 채무완제 불가 → 각 사원 연대책임
- ② 회사재산에 강제집행 주효하지 않음 → 각 사원 연대책임
- ③ 회사에 변제자력 + 집행 용이 → ①② 적용 ✕
- ※ **사원들끼리의 연대**이지, 회사-사원 간 연대 ✕ (회사-사원 간에는 순서 있음)

#### (2) 직접책임 vs 간접책임

| 구분 | 내용 |
|------|------|
| **직접책임** (합명 사원) | 회사채권자에게 직접 책임 |
| **간접책임** (주주) | 회사에 대해 주식인수대금 납입의무만 → 채권자에게 직접 책임 ✕ |

#### (3) 사원의 항변
- ① 회사가 주장할 수 있는 항변으로 채권자에게 대항 가능
- ② 회사에게 상계·취소·해제 권리 있으면 → 사원도 변제 거부 가능

#### (4) 특수한 사원의 책임

| 유형 | 핵심 |
|------|------|
| **신입사원** (§213) | 가입 **전** 회사채무에도 동일한 책임 |
| **퇴사원** (§225) | 퇴사등기 전 채무 → 등기 후 **2년 내** 동일 책임 (지분양도사원도 동일) |
| **자칭사원** (§215) | 사원 아닌 자가 사원으로 오인시킴 → 오인 거래자에 대해 사원과 동일 책임 (**외관신뢰**) |

---

### C1-2. 출자

#### (1) 노무/신용출자
- 가능 → 단, **무한책임사원만** → **"무무!!"**
- 설립등기 전 출자 완료 불요 / 퇴사 시 노무출자도 환급

#### (2) 채권출자
- 가능 → 미변제 시: **채**권액 변제 + **이**자 지급 + **손**해배상 → **"채이손!!"**

---

### C1-3. 기관의 구성 — 자기기관의 원리

| 구분 | 자기기관 (인적회사) | 소유-경영 분리 (물적회사) |
|------|-------------------|----------------------|
| 주된 의미 | 사원 = 당연히 집행권·대표권 | 이사로 선임되어야 집행권·대표권 |
| 부수적 의미 | 사원 아닌 자 → 일꾼 될 수 없음 | 사원 아닌 CEO도 가능 |

#### (1) 집행권

**§200 업무집행의 권리·의무**
- ① 각 사원 → 업무집행 권리/의무 (정관에 다른 규정 없는 한)
- ② 다른 사원 이의 → 행위 중지 → **총사원 과반수 결의**

**§201 업무집행사원**
- ① 정관으로 1인/수인을 업무집행사원 지정 → 그 사원만 집행
- ② 다른 업무집행사원 이의 → **업무집행사원 과반수 결의**

#### (2) 대표권

**§207 대표권**
- ① 업무집행사원 미지정 → **각 사원이 대표**
- ② 복수 업무집행사원 → **각 업무집행사원이 대표**
- ③ 정관/총사원 동의 → 대표사원(사장) 특정 가능

**§208 공동대표**
- ① 정관/총사원 동의 → 복수 공동대표 가능
- ② 제3자의 회사에 대한 의사표시 → 공동대표 중 **1인에게** 하면 됨 (**수동대표**)

---

### C1-4. 경업과 자기거래 ★★★

> 대상: **모든 사원** (집행권/대표권 유무 불문)

#### (1) 경업금지 (§198) — 다른 사원 **전원의 동의**

| 금지 유형 | 내용 | 위반 시 |
|----------|------|---------|
| **거래금지** | 자기/제3자 계산으로 회사 영업부류 거래 불가 | **개손제** (개입권/손배/제명) |
| **겸직금지** | 동종영업 다른 회사의 무한책임사원/이사 겸직 불가 | **손제** (손배/제명) |

#### (2) 자기거래금지 (§199) — 다른 사원 **과반수의 결의**
- 자기/제3자 계산으로 회사와 거래
- 경업 피해 > 자기거래 피해 → 경업은 전원동의, 자기거래는 과반수

#### (3) 합명회사 사원의 제명사유
- ① 출자의무 불이행
- ② 경업금지의무 위반

---

### C1-5. 지분의 양도
- **다른 사원 전원의 동의** 필요 (§197)
- 이유: 인적 신용 기초 → 사원 변동이 전원의 책임·신용에 직접 영향

### C1-6. 상속
- **원칙: 상속 ✕** → 사망 = 퇴사(지분환급) → out!!
- **예외**: ① 정관에 규정 / ② 회사해산 후 사원 사망
- ★ 물주의 지위 = 상속 ○ / **일꾼의 지위 = 상속 ✕**

### C1-7. 퇴사 (= 지분환급)

> 물적회사(주식·유한) → 퇴사제도 없음 (지분양도 자유)
> 합명·합자·LLC → 퇴사제도 있음

| 구분 | 내용 |
|------|------|
| 임의퇴사 (원칙) | 존립기간 미정/종신 → 영업연도 말 퇴사 (6월 전 예고) |
| 임의퇴사 (예외) | 부득이한 사유 → 언제든지 퇴사 |
| **법정퇴사** | 사원 의사 무관 → ★★★★★ **사/금/파/제명** |

#### 지분압류채권자의 퇴사청구 (§224)
- 사원 지분 압류한 채권자 → 영업연도 말에 그 사원 퇴사 청구 가능 (6월 전 예고)
- 경제적 가치: ① 매 연도 이익배당 권리 ② 출자환급 권리

#### 퇴사의 부수적 효과 — 상호변경청구권 (§226)
- 퇴사 사원 성명이 회사 상호에 사용된 경우 → 사용 폐지 청구 가능

## C2. 합자회사 (익명조합과 유사)

> 정관에 각 사원의 무한/유한 책임 구분을 **절대적 기재**

### C2-1. 정관변경 (유한 ↔ 무한)

| 변경 방향 | 처리 방식 |
|----------|----------|
| 유한 → 무한 | **신입사원**의 책임과 동일 |
| 무한 → 유한 | **퇴사원**의 책임과 동일 |

### C2-2. 특수 책임
- (가) **자칭무한책임사원**: 유한이 무한으로 오인시킴 → 오인 거래자에 대해 무한과 동일 책임
- (나) **합명→합자 조직변경**: 무한→유한 된 자 → 조직변경 등기 **전** 채무에 대해 등기 후 **2년 내 무한책임** (= 퇴사원)

---

### C2-3. 무한책임사원

| 항목 | 내용 |
|------|------|
| 사원의 책임 | 무한/직접/연대 — 합명회사 준용 (§269) |
| 지분양도 | 다른 사원 **전원**(무한+유한)의 동의 (§197) |
| 경업 | 다른 사원 **전원**의 동의 |
| 자기거래 | 다른 사원 **과반수** 결의 |

---

### C2-4. 유한책임사원 (= 물주)

#### (1) 권한

| 권한 | 유무 | 비고 |
|------|:----:|------|
| 집행권 | ✕ | 임의규정 (정관으로 부여 가능) |
| 대표권 | ✕ | **강행규정** (부여 불가) |
| 감시권 | ○ | 원칙: 영업연도 말 / 예외: 중요 사유 → 법원 허가로 언제든지 |

#### (2) 사원의 책임
- **유한 / 직접 / 연대**
- 유한 한도 = 출자가액 중 **미이행 부분** + 이익 없이 배당받은 금액(**불법배당**)

#### (3) 지분양도
- **무한책임사원 전원의 동의** (§276) ← 사원 전원까지는 불요
- 이유: 유한사원은 경영권·대표권 없고 책임 제한 → 지분양도가 회사 신용 구조를 크게 흔들지 않음

#### (4) 상속
- **상속 ○** — 상속인이 지분 승계하여 사원 (법정퇴사사유 = 사/금/파+제명)

#### (5) 경업과 자기거래
- 경업: **금지 ✕** (지위 악용 위험 없음)
- 자기거래: **금지 ✕** (지위 악용 위험 없음)

#### (6) 노무/신용출자
- **✕** — 무한책임사원만 가능

### C4-5. 유한회사 사원의 추가책임 ★★

| #   | 상황                                 | 책임자               |        면제        |
| --- | ---------------------------------- | ----------------- | :--------------: |
| ①   | 회사설립 시 현물출자/재산인수 **과대평가** (§550)   | 사원 (연대)           |        불가        |

| ③   | 자본금 증가 시 현물출자/재산인수 **과대평가** (§593) | 동의한 사원 (연대)       |        —         |
| ④   | 유한→주식 **조직변경** (§607)              | 사원 + 이사 + 감사 (연대) | 사원=불가 / 이사·감사=가능 |

**추가책임 정리**
- 1단계 — 회사설립: §550(과대평가) + §551(출자미필)
- 2단계 — 자본금 증가: §593(과대평가)
- 3단계 — 조직변경: §607
- ※ **사원+이사+감사** 3자 책임: §551(출자미필), §607(조직변경)
