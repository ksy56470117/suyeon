# 🃏 두문자 키워드 플래시카드

> **Spaced Repetition** 플러그인 사용법:
> 1. 옵시디언 설정 → 커뮤니티 플러그인 → "Spaced Repetition" 검색 → 설치
> 2. 이 노트를 열고 왼쪽 리본의 카드 아이콘 클릭 → 복습 시작
> 3. `::` 앞이 질문, 뒤가 답입니다

## 상행위총론

상사유치권 적용요건 5가지 #card
:: **당피목관반** — 당사자 쌍방 상인 · 피담보채권 상행위로 인한 것 · 목적물 점유 · 채권과 목적물 관련성 불요(상사특칙) · 반환거부 가능

상행위 특칙 중 상사매매 4단계 #card
:: **공탁경매 → 확정기매매 → 검사통지 → 보관공탁경매**

상호계산 불산입 항목 #card
:: **채이손!!** — 채권양도 · 이중지급 (어음할인) · 손해배상청구권

## 회사법 통칙·구조조정

회사법상 합병 절차 순서 #card
:: **계대결채보등** — 계약 · 대차대조표 등 공시 · 결의(주총 특별) · 채권자보호절차 · 보고총회(or 서면) · 등기

주식매수청구권이 인정되는 경우 #card
:: **구영승** — 구조조정(합병·분할·교환·이전) · 영업양도 · 승인(양도제한주식)

종류주주총회가 필요한 경우 #card
:: **정구회** — 정관변경으로 손해 · 구조조정(합병·분할) · 회사채 발행(전환사채·신주인수권부사채 중 종류주주에게 불이익)

악법 3종 (반대주주 보호 없음) #card
:: 간이합병(피) · 소규모합병(존) · 소규모분할합병

## 주식·주주

종류주식 중 의결권 제한 종류주식의 발행한도 #card
:: **종자특감** — 종류주식 · 자본 1/4 · 특별결의 · 감축가능

주권의 선의취득 요건 #card
:: **비설권 유가증권** — ① 주권 점유 ② 배서 연속(기명주권) or 교부(무기명) ③ 선의·무중과실

주주명부 폐쇄 vs 기준일 #card
:: 폐쇄: 일정 기간 명의개서 정지 (3개월 이내) / 기준일: 특정일 기준 주주 확정 (2주 전 공고) / **병용 가능**
