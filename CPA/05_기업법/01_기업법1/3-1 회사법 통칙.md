## 제1절 회사의 개념

### 제1관 회사의 의의

#### 회사의 의의 (§169)

회사란 상행위(22가지)나 그 밖의 영리를 목적으로 하여 설립한 법인
⇒ **상사회사(22가지) / 민사회사**

> 2011년 상법 개정 때 **사단**은 요건에서 빠짐

##### 영리
- 이윤추구활동 후 이익을 **사원에게 분배**
- 사원 = 지분을 갖고 있는 자

##### 사단 (~~삭제됨~~)
- 단체성이 강한 모임
- cf. **조합** = 단체성이 약한 모임
- 물적회사는 1인 회사가 가능 → 사단이라는 조건이 불필요해짐
- 인적회사의 내부관계에는 민법의 조합규정을 적용

##### 법인
- 사원과 회사는 남남 = **책임의 분리**
- **본점소재지에서 설립등기한 때** 법인 성립
- 법인의 분류
	- 영리법인 : 회사
	- 비영리법인
		- 공익법인 : 재단법인(예:학교법인, 의료법인) / 비영리사단법인
		- 일반공법인 (예:서울특별시)

> ★ 정관은 법원(法源)이지만 약관은 법원 아님

### 1인회사

사원이 1명만 / 회사의 종류별로 1인회사 인정여부 차이 있음

| 구분 | 내용 |
|------|------|
| **인적 (합명/합자)** | 설립할 때 2명 이상이 정관작성(1인 불가) / 사원이 1명만 남으면 해산사유 (1인 존속 불가) |
| **주식/유한/LLC** | 1인 가능 (1인 설립 & 1인 존속 모두 가능) |

##### LLC의 특징
- **내부관계** : 합명회사(인적회사)에 가까움 → 민법의 조합 준용
- **외부관계** : 물적회사 성격 (사원이 아닌 사람도 업무집행자 가능)

### 주주총회 운영상의 하자

#### 주주총회 소집절차
이사회 결의로 총회소집 ⇒ 대표이사가 소집집행 ⇒ 주주에게 소집통지(2주 전까지 서면/전자문서) 발송 ⇒ 통지할 때 총회의 목적사항 기재

#### 주주총회 소집절차의 하자 ★★★★★

| 하자 내용 | 소의 종류 |
|-----------|-----------|
| ① 이사회 결의로 총회소집 (경미한 하자) | 총회결의취소의 소 |
| ② 대표이사가 소집집행 (없이 진행) | 총회결의부존재확인의 소 |
| ③ 주주에게 소집통지 - 일부에게만 통지 X | 총회결의취소의 소 |
| ③ 주주에게 소집통지 - 대부분에게 통지 X | 총회결의부존재확인의 소 |
| ④ 통지할 때 총회의 목적사항 기재 X | 총회결의취소의 소 |

#### 1인회사 - 주주총회 하자
- 1인회사의 경우 ①②③④(주총 소집절차) 중 어떤 걸 위반했더라도 **1인주주의 의사에 합치한다면** (= 1인주주의 뜻에 따라 총회의사록이 작성되었다면) **총회결의로서 유효**
- 총회소집절차에 관한 규정 중 복수의 주주를 전제로 한 규정들은 1인회사에 적용 X

#### 1인회사 - 자기거래
- **1인주주라면 이사회 승인 필요없음**

### 이사의 자기거래제한의 위반

이사회 승인 받지 않고 거래했다면? → **상대적 무효**

| 구분 | 효력 |
|------|------|
| 내부적 | 무효 |
| 외부적 - 선의 | 유효 |
| 외부적 - 악의 | 무효 |

**★★★ 판례**
- **주주 전원의 동의** → 주주총회에서 승인 가능
- **정관에 규정 있음** → 나중에 이사회승인이 없었다는 이유로 회사가 책임회피 불가능

#### 기타 관련문제
- 1인회사가 인정된다고 해도 **회사와 주주는 별개의 인격**
- 1인주주가 회사의 재산 감소 → 횡령죄/배임죄
- 매매 자체를 유효하게 인정하는 것이지 횡령/배임이다.

## 법인격부인론

### 개념
회사의 법인격 남용 ⇒ 법인격을 **박탈하지는 않고** 회사와 사원의 인격을 동일시하여 회사채권자를 보호하려는 이론
⇒ **회사채무를 사원(지배주주)에게 전가**

> 회사 자금과 개인 자금을 심하게 뒤섞어 알맹이도 없게 됨
> → ★★ 회사와 주주의 독립성이 사실상 사라짐 (그렇다고 법인격이 박탈되는 건 아님)

### 판례
회사가 실질적으로는 배후자(=지배주주)의 개인기업에 불과하거나, 배후자에 대한 법률적용을 회피하기 위한 수단으로 쓰이는 경우
⇒ **민법 §2**에 따라 회사의 행위에 대해 배후자에게도 책임을 물을 수 있음

### 적용요건
1. 지배주주의 개인사업체처럼 운영
2. 지배주주에 대한 법률적용을 회피하기 위한 수단으로 운영

### 적용효과
1. 법인격부인론 적용되더라도 **회사의 법인격은 박탈되지 않음**
2. 회사의 채무를 지배주주가 변제할 책임 부담 (회사의 책임이 소멸하지도 않음)
3. 회사에 대한 판결의 효력이 지배주주에게 미치지 않음
	- → 채권자가 회사에 대한 승소판결 → 지배주주에게 강제집행 불가
	- → **회사와 지배주주를 공동피고로 제소** → 각자에 대한 집행권원(판결문) 취득해야 함

### 역적용의 문제 - 사해설립 (Reverse Piercing)

법인격부인론의 일반적 개념 : 회사의 채무를 지배주주에게 전가
↔ **역적용** : 설립자의 채무를 회사에 전가

#### 사해설립 시 일반적 구제수단
① 임이 가진 A회사 주식에 대한 강제집행
② **상법 §176 회사해산명령** → 적용 불가 (공익적 규정이어서 사적채권자 이용 불가)
③ **상법 §185 사해설립취소 소** → 사실상 적용 거의 불가 (합명·합자·유한·LLC에만 있고 **주식회사엔 없음**)
→ 그래서 사기꾼들은 주식회사를 선호

#### 법인격부인론의 역적용 (④)
④ **판례 인정!** (안건사 사건)
- 기존회사(주안건사) 재산을 신설회사(주안건미디어)로 이전한 경우
- 대법원: "기존회사 채권자는 기존회사와 신설회사 어느 쪽에 대해서도 이행을 청구할 수 있다."
- → 역적용을 통해 신설회사 재산에도 집행 가능

## 제2절 회사의 종류

## 1. 합명회사

**사원의 책임 : 무한 / 직접 / 연대책임 (순서 불문, 비율 불문)**

### 사원의 책임

#### 합명회사 사원책임 (§212)
1. 회사재산으로 채무완제 불가 ⇒ 각 사원 연대책임
	- (회사-사원 연대가 아니라, **사원들끼리의 연대** ← 회사-사원 간에는 순서가 있음)
2. 회사재산에 강제집행 주효하지 않음(= 효과를 못봤다) ⇒ 각 사원 연대책임
3. 회사에 변제자력 있음 + 집행용이 ⇒ 1), 2) 적용 X

#### 직접책임 vs 간접책임
- **직접책임** (합명회사 사원) : 회사채권자에게 직접 책임
- **간접책임** (주주) : 주주는 회사에 대해 주식인수대금 납입의무만 있음 → 회사채권자에 대해서는 책임지지 않음

#### 사원의 항변
1. 사원이 회사채무에 관하여 변제의 청구를 받은 때 → **회사가 주장할 수 있는 항변**으로 채권자에게 대항 가능
2. 회사가 채권자에 대해 상계·취소·해제할 권리가 있는 경우 → **사원도 변제를 거부**할 수 있음

#### 특수한 사원의 책임
| 유형 | 내용 |
|------|------|
| **신입사원 책임 (§213)** | 합명회사 성립 후 가입한 사원은 가입 전에 생긴 회사채무에 대해 다른 사원과 동일한 책임 |
| **퇴사원 책임 (§225)** | 본점소재지에서 퇴사등기 하기 전에 생긴 회사채무 → 등기 후 **2년 내**에는 다른 사원과 동일한 책임 (지분양도사원도 동일) |
| **자칭사원 책임 (§215)** | 사원 아닌 자가 타인에게 자기를 사원으로 오인시키는 행위 → 오인으로 인해 합명회사와 거래한 자에 대해 사원과 동일한 책임 (외관신뢰) |

### 출자
- **노무/신용출자** 가능 (설립등기 전에 출자 완료할 필요 없음, 퇴사 시 노무출자도 환급)
	- 노무/신용은 **무한책임을 지는 사원만** 가능 → **무무!!**
- **채권출자** 가능
	- 그 채권이 변제되지 않으면 : 채권액 변제 + 이자 지급 + 손해배상 → **채이손!!**

### 기관의 구성 - 자기기관의 원리

| 구분 | 자기기관의 원리 (인적회사) | 소유와 경영의 분리 (물적회사) |
|------|--------------------------|------------------------------|
| 주된 의미 | 사원이기만 하면 당연히 집행권/대표권 있음 | 사원(주주)는 이사로 선임된 뒤에만 집행권/대표권 있음 |
| 부수적 의미 | 업무집행사원을 둔 때에도 사원 아닌 자가 일꾼 될 수 없음 | 사원(주주) 아닌 CEO도 물적회사에서 일꾼 가능 |

#### 집행권 (§200, §201)

**업무집행의 권리·의무 (§200)**
1. 각 사원은 정관의 다른 규정이 없다면 회사 업무 집행할 권리/의무 있음
2. 다른 사원의 이의 있음 ⇒ 행위 중지 ⇒ **총사원 과반수의 결의**

**업무집행사원 (§201)**
1. 정관으로 사원의 1인/수인을 업무집행사원으로 정함 ⇒ 그 사람(들)만 업무집행 권리/의무
2. 다른 업무집행사원의 이의 ⇒ **업무집행사원 과반수 결의**

#### 대표권 (§207, §208)

**대표권 (§207)**
- 정관으로 업무집행사원 정하지 않음 ⇒ 각 사원이 대표
- 여러 명의 업무집행사원 정함 ⇒ 각 업무집행사원이 대표
- 정관/총사원 동의 ⇒ 업무집행사원 중 특히 회사를 대표할 자(= 대표사원/사장) 정할 수 있음

**공동대표 (§208)**
1. 정관/총사원 동의 ⇒ 여러 명 공동대표 가능
2. 제3자의 회사에 대한 의사표시는 **공동대표의 권한 있는 1인**이 해도 됨 (= 수동대표)

### ★★★ 경업과 자기거래 (대상 : 모든 사원 - 집행권/대표권 유무 따지지 않음)

#### 경업금지 (§198) - 다른 사원 **전원의 동의**
모든 사원은 <다른 사원> 전원의 동의가 없으면
- **거래금지** : 자기/제3자의 계산으로 회사의 영업부류에 속하는 거래 불가
	- 위반 시 → 개손제 (실질적 개입권 / 손배청구 / 제명)
- **겸직금지** : 동종영업을 목적으로 하는 다른 회사의 무한책임사원(일꾼)/이사가 되지 못함
	- 위반 시 → 손제 (손배청구 / 제명)

#### 자기거래금지 (§199) - 다른 사원 **과반수의 결의**
사원은 "다른" 사원 "과반수"의 결의가 있을 때에 한해 자기/제3자의 계산으로 회사와 거래 가능
← 일반적으로 경업의 피해가 자기거래의 피해보다 더 크다

#### 합명회사 사원의 제명사유
- 출자의무 불이행 (돈 안 내기)
- 경업피지의무 (뒤통수) 위반

### 지분의 양도
- **다른 사원 전원의 동의** 필요 (§197)
	- 이유 : 합명회사는 사원들의 인적 신용을 기초로 하며, 새로운 사원이나 지위변동은 모든 사원의 책임·신용에 직접 영향을 미치므로

### 상속여부
- **원칙** : 상속 X (합명회사 사원의 사망 ⇒ 퇴사 - 지분환급 ⇒ out!!)
- **예외**
	1. 정관에 규정이 있는 경우 (미리 정해둠)
	2. 회사해산 후에 사원이 사망한 경우

> Q) 합명회사 사원의 지위는 원칙적으로 상속되지 않는다 (O)
> = 물주의 지위는 상속되지만, **일꾼의 지위는 상속되지 않는다**

### 퇴사 (= 지분환급)
물적회사는 퇴사제도가 없음
→ 지분양도가 자유로운 **주식/유한회사에는 퇴사제도가 없고**, 합명/합자/LLC에만 있음

| 구분 | 내용 |
|------|------|
| **임의퇴사 원칙** | 정관으로 존립기간 미정/종신까지 존속 ⇒ 영업연도 말에 퇴사 가능 (6월 전 예고) |
| **임의퇴사 예외** | 사원이 부득이한 사유 ⇒ 언제든지 퇴사 가능 (익명조합 계약의 해지) |
| **법정퇴사** | 사원의 의사와 무관한 퇴사 ⇒ ★★★★★ **사/금/파/제명** |

#### 지분압류채권자의 퇴사청구 (§224)
- 사원의 지분을 압류한 채권자는 영업년도 말에 그 사원 퇴사 가능
- (회사/사원에 대해 6월 전 예고)
- 채권자가 사원의 지분 압류 ⇒ 경제적 가치 : ①채무자가 매 연도에 이익배당받을 권리 ②출자환급권리

#### 퇴사의 부수적 효과 - 상호변경청구권 (§226)
퇴사한 사원의 성명이 회사의 상호 등에 사용된 경우에는 그 사원은 회사에 대해 그 사용의 폐지를 청구할 수 있음

## 2. 합자회사 (익명조합과 비슷)

합자회사의 경우 정관에 각 사원이 무한책임/유한책임 중 어떤 책임을 지는지 **정관에 절대적으로 기재**해야 함

### 정관변경 (유한 ↔ 무한)
| 변경 | 처리 방식 |
|------|-----------|
| 유한책임사원 → 무한책임사원 | 신입사원의 책임과 동일하게 처리 |
| 무한책임사원 → 유한책임사원 | 퇴사원의 책임과 동일하게 처리 |

- **자칭무한책임사원** : 유한책임사원이 타인에게 무한책임사원으로 오인시킬 행위를 한 때 → 오인으로 인해 회사와 거래한 자에 대해 무한책임사원과 동일한 책임
- **합명→합자회사 조직변경** : 무한책임사원이 유한책임사원으로 된 자는 본점에 조직변경 등기 하기 전에 생긴 회사 채무에 대해 등기 후 **2년 내에는 무한책임** 부담 (= 퇴사원)

- **지분의 양도** : 다른 사원 전원의 동의 필요 (§197) (유한+무한 전원)
- **경업** : 다른 사원 전원의 동의
- **자기거래** : 다른 사원 과반수의 결의

- 대표권 X (강행규정)

- **집행권** (임의규정) X - 정관에 규정이 있다면 바쁘면 일 좀 도와줄 수 있음

## 4. 유한회사 (소규모 물적회사)

| 주체 | 경업 | 자기거래 |
|------|------|----------|

| **이사(일꾼)** | 사원총회 승인 | ①감사(임시기관) 승인 → ②사원총회 승인 |

### 노무/신용출자
- X : 무한책임사원만 노무/신용출자 가능 → 사원(유한책임)은 노무/신용출자 불가능

> **논리 충돌 주의** (그러나 아래 문장들은 모두 맞는 선지)
> - 주식회사의 사원성명은 정관기재사항 X, 나머지(유한회사 포함)는 정관기재사항임
> - 구법) 유한회사의 지분양도에 사원총회특별결의(70% 찬성) 필요 → 신법) 자유롭게 양도 가능
> - 유한회사 정관 바꾸려면 사원총회특별결의 필요
> - 유한회사의 사원성명은 정관에 적어야 함
> - 상법상 유한회사의 지분을 양도할 때 사원총회의 특별결의 거칠 필요 없음
> ※ 논리충돌 : 자유롭게 지분양도 ⇒ 사원성명이 바뀜 ⇒ 정관 바꿔야 함 ⇒ 사원총회특별결의해야 함

### 유한회사의 특징

물적회사이긴 하지만 주식회사보다 소규모 → 국가에서 관리를 빡세게 안 함
유한회사의 사원은 출자금액한도로 유한책임지는 것이 **원칙** / 예외적으로 추가적인 책임을 부담하는 경우도 다수 있음

#### 유한회사 사원의 추가책임

| 상황 | 책임자 | 내용 |
|------|--------|------|
| **회사설립 시 현물출자/재산인수 과대평가 (§550)** | 사원 | 실가가 정관에 정한 가격에 현저하게 부족한 때 → 회사성립 당시의 사원이 연대하여 부족액 지급 책임 (면제 불가 - 자본충실이념) |
| **출자미필에 대한 책임 (§551)** | 사원+이사+감사 | 회사성립 후 출자금액 납입 또는 현물출자 이행 미완 → 성립 당시의 사원+이사/감사의 책임 (이사와 감사의 책임은 면제 가능) |
| **자본금 증가 시 현물출자/재산인수 과대평가 (§593)** | 사원 | 자본금 증가 당시의 실가가 결의에 의해 정한 가격에 현저하게 부족한 때 → 그 결의에 동의한 사원이 연대하여 부족액 지급 책임 |
| **유한회사→주식회사 조직변경 (§607)** | 사원+이사+감사 | 주식 발행가액의 총액 부족 → 조직변경 결의 당시의 유한회사의 사원/이사/감사가 연대하여 부족액 지급 책임 (사원책임-면제불가 / 이사감사책임-면제가능) |

#### 추가책임 정리
1. 회사설립 : 현물출자/재산인수의 과대평가(§550) / 출자미필(§551)
2. 자본금 증가 : 현물출자/재산인수의 과대평가(§593)
3. 유한회사 → 조직변경 : 조직변경(§607)
	- ※ 출자미필(§551), 조직변경(§607) : 사원+이사+감사

## 6. 합자조합 ← 법인이 아님 (상행위법 적용 / 회사법 적용 X)

### 개념
합자조합은 **무한책임조합원 + 유한책임조합원**으로 구성되는 공동기업형태
- 상법의 편제상 회사법이 아니라 **상행위**에 배치 (왜냐면 회사가 아님)
- 합자조합은 조합원 간의 계약관계일 뿐 회사가 아니므로 **설립등기를 한 때가 아니라 조합계약을 맺은 때에 성립**

> **법인이 아니므로** : 정관 → 조합계약 / 대표 → 대리 / 본점 → 주된 영업소 / 퇴사 → 탈퇴

### 등기
- 합자조합의 무한책임조합원은 조합설립 후 **2주 내**에 주된 영업소의 소재지에서 조합의 목적과 명칭, 무한책임조합원의 성명, 조합원의 출자의 목적/가액 등기
- 예외 : 유한책임조합원이 업무집행하는 예외적 경우 → 유한책임조합원의 성명도 등기
- **등기가 조합설립의 효력발생요건은 아님**

| 등기기간 비교 | 기간 |
|--------------|------|
| 총칙/상행위 | 설립 후 지체없이 |
| 회사법 | **2주 내** |

### 업무집행조합원 (무한책임조합원)

- **사원의 책임** : 무한 / 직접 / 연대 (합명회사 규정 준용)
- **노무/신용출자** 가능
- **기관의 구성** : 자기기관의 원리 (집행권+대리권 O)

#### ★★★ 경업과 자기거래
- **경업** : 다른 조합원 전원의 동의
- **자기거래** : 다른 조합원 과반수의 결의

#### 지분의 양도
- **다른 조합원 전원의 동의** 필요

#### 업무집행 원칙
1. 업무권+대리권 O
2. 선량한 관리자의 주의
3. 둘 이상의 업무집행조합원(#1, #2) → #1이 추진하는 일에 대해 #2가 이의 → 그 행위 중지 → **업무집행조합원 과반수의 결의**에 따라야 함

---

### 유한책임조합원 (← 물주)

- **사원의 책임** : 유한 / 직접 / 연대
- 유한책임(한도) = 출자가액에서 미이행한 부분 + 이익없이 배당받은 금액(불법배당)
- **노무/신용출자** 불가능

#### 기관의 구성 - 경영참여권의 배제
- **원칙** : 집행권, 대리권 X / 감시권 O
- **예외** : 조합계약으로 다르게 정할 수 있음

#### ★★★ 경업과 자기거래
- **경업** : 금지 안 함 (지위 악용 위험 X)
- **자기거래** : ★★★ 금지 O (다른 조합원 과반수의 결의)

#### 지분의 양도
- ★★★ **양도(권리의무의 승계)여부는 조합계약에서 정하는 바에 따름**

#### 상속
- **상속 O** : 상속인이 지분을 승계하여 사원이 됨 (= 사/금/파+제명)
