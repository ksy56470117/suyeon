# Part A. 조직변경

## A1. 의의
- 법인격 동일성 유지 + 성질 유사(인적↔인적, 물적↔물적) 다른 종류로 변경

## A2. 가능한 조직변경
- 합명 ↔ 합자 / 주식 ↔ 유한 / 주식 ↔ LLC / 유한 ↔ LLC (**✕**)

### ★★ 조직변경 후 책임 (§244)
- 합명→합자 유한책임사원 된 경우: 본점등기 전 채무 → 등기 후 **2년 내 무한책임**
- 합자→합명: 채권자보호규정 ✕ / 합명→합자: 채권자보호규정 ○

## A3. 절차

| 항목 | 내용 |
|------|------|
| 내부 | 원칙: 종류·시기 불문 **총사원(총주주) 동의** / 예외: 유한→주식 정관정함 → 사원총회특별결의 |
| 외부 | 주식-유한/LLC 간 **채권자보호절차** |
| 효력 | 법인격 동일성 유지 → 승계/이전 ✕ |

# Part B. 합병

## B1. 의의
- 2개↑ 회사 합침 → 최소 1개 해산·소멸 / **청산절차 거치지 않음** / 포괄승계

## B2. 합병의 종류 — 주식회사

| 구분 | 결의 |
|------|------|
| 원칙 | 주주총회**특별결의** |
| 예외 | **간이/소규모합병** → 이사회결의 |

### 간이합병 vs 소규모합병 ★★★★

| | 간이 | 소규모 |
|---|------|--------|
| 공통 | **흡수합병에만** 가능 |
| 차이 | 소멸회사(B) 총회 → 이사회 | 존속회사(A) 총회 → 이사회 |
| 주매청 | ○ | **✕ (악법3종set)** |
| 채보절차 | ○ | ○ |

> ★★★ **악법3종set**: 소규모(합병/분할합병/주식교환) → 주매청 ✕

## B3. 절차 (**계·대·결·채·보·등**)

| 단계 | 내용 |
|------|------|
| **계** | 합병계약서 작성 (법정기재사항 흠결 → 무효소의 원인) |
| **대** | 계약서/대차대조표 공시 (총회2주전~합병일+6월) |
| **결** | 주총특결 (간이/소규모→이사회) / 종류주총 / **주매청** |
| **채** | 채권자보호절차 (결의후 2주내 공고+개별최고 / 이의 1월↑) |
| **보** | 보고총회(흡수)/창립총회(신설) — 이사회 공고로 대체 가능 |
| **등** | 합병등기 = **효력요건** (창설적효력 ○) |

### 주매청 행사조건 → [참조: 3-4 주주총회](./3-4_1절_주주총회_정리노트.md)
- 이사회 결의 후 총회 전까지 서면 반대의사 → 총회결의일+20일 내

## B4. 합병대가
- (1)신주발행 (2)자기주식이전 → 사원지위 수용 / (3)금전 (4)현물 (5)모회사주식=삼각합병

## B5. 합병무효의 소 (§529)
- 원고: 주주/이사/감사/청산인/파산관재인/**불승인채권자**
- 제소기간: 합병등기일+**6월**
- 무효판결 후: 재산=**공유** / 채무=**연대책임**

### 합병 vs 영업양도 비교

| | 합병 | 영업양도 |
|---|------|--------|
| 주체 | 회사만 | 자연인도 |
| 승계 | **포괄** | 특정(개별이전) |
| 일부가능 | ✕ | ○ |
| 채보절차 | ○ | ✕ |
| 무효의 소 | ○ | ✕ |
| 공통 | 총회특결 / 반대주주 주매청 |

# Part C. 분할

> 분할은 **주식회사만** 가능

## C1. 종류
- 단순분할 / 분할합병(흡수/신설) / 소멸분할 vs 존속분할 / 인적분할 vs 물적분할

## C2. 절차 — 단순분할 vs 분할합병

| 항목 | 단순분할 | 분할합병 |
|------|---------|---------|
| 채보절차 | 원칙 ✕ (예외: 분할채무결의 → ○) | **○** |
| 주매청 | **✕** | ○ (예외: 소규모분할합병 ✕) |

## C3. 관련회사 연대책임 (§530의9)
- 원칙: 분할회사·신설회사 **연대책임**
- 예외: 총회특결로 **분할책임결의** 가능 → 각자 분담

## C4. 분할무효의 소
- 합병무효의 소 준용 / 분할등기일+6월

# Part D. 주식의 포괄적 교환·이전

> 물적요소 변동 없이 주주구성만 변경 → **채보절차 불필요**

## D1. 절차 (**계·대·결·실·등**)
- 계약서/계획서 작성 → 주주에게 정보제공(not 채권자) → 총회특결 → 주권실효절차 → 등기

### 간이/소규모 교환 (간이/소규모 **이전은 없음**)

## D2. 교환 vs 이전 효력

| | 주식교환 | 주식이전 |
|---|--------|--------|
| 효력발생 | **교환의 날** | **이전등기 한 때** |
| 등기 창설적효력 | ✕ | ○ |
| 무효판결 → 모회사 | 법인격 유지 | **청산(법인격 소멸)** |

### 무효의 소
- 기간: 교환/이전일+6월
- ★★ 원고(5): 주주/이사/감사/감사위원/청산인 (**불승인채권자 ✕**)
