# Part A. 주식

## A1. 분류
- 기명/무기명: 현행법 무기명 **폐지** (무기명사채는 가능)
- 액면/무액면: 무액면주 발행 → 정관근거 필요 / 전환 → 회사만(정관따라) / 자본금 변경 ✕

## A2. 주식의 분할 (§329-2)
- 총회특결 / 액면주 100원 미만 불가 / 주권실효절차(병합준용) / 자본금·순자산 불변

## A3. 주식의 병합 (감자수단)
- 주권실효절차(윽박) + **채권자보호절차** 필요 / 단주처리: 경매(원칙)/거래소/법원허가

## A4. 질권의 물상대위 (§339)
- 소각/병합/분할/전환 시 → 종전 주주가 받을 금전·주식에 질권 행사 가능

# Part B. 주주

## B1. 타인명의 주식인수 (§332)
- 납입책임: ①가설인/②무승낙 → 실제인수자 / ③승낙有 → **연대납입**
- 주주 판단: 규정 ✕ → 판례(2017): **주주명부 기준**

## B2. 주주권 분류

| | 단독주주권 | 소수주주권 |
|---|---|---|
| 자익권(money) | **전부** 단독 | ✕ |
| 공익권(power) | 일부 | 일부 |

### 소수주주권 비율

| 비율 | 권리 (일반) |
|------|----------|
| 1/100 | 위법행위유지청구·대표소송·검사인선임청구 |
| 3/100 | 대부분의 소수주주권 |
| 10/100 | only 해산판결청구 |

### 상장회사 특례 (더 낮은 지분 + **6개월↑** 보유) → 임.제.열.해.검.유.대

## B3. 주주평등원칙
- 일반규정 ✕ / 의결권·신주인수권·이익배당 등 개별규정 ○ (모두 **강행규정**)

# Part C. 종류주식

## C1. 이익배당/잔여재산분배 종류주식
- 우선주·후배주·혼합주 / 참가적/비참가적 / 누적적/비누적적

## C2. 의결권 없는 종류주식 ★★★★★

### 의결권 제한 (법률로만 가능) — **종.자.특.감**

| 제한 | 범위 | 정족수 계산 |
|------|------|----------|
| **종** (종류주식) | 정관으로 정하기 나름 | 발행주식총수 **불산입** |
| **자** (자기주/상호주) | 전면적 부정 | 발행주식총수 불산입 |
| **특** (특별이해관계인) | 개별적 부정 | 출석의결권수 불산입 |
| **감** (감사선임 3/100초과) | 개별적 부정 | 출석의결권수 불산입 (판례: 발행총수도 불산입) |

### 무의결권주주 권리
- 의결권 ✕ + 통.제.집 ✕
- 예외(의결권 ○) = **종.전.정.립.분**: 종류총회/전원동의(조직변경·책임면제)/정관부활조건/창립총회/회사**분**할

## C3. 상환종류주식 (§345)
- **배당가능이익**으로 소각 (자본충실) / 종류주식만 대상 / 보통주 ✕
- 상환 → 발행주식수 감소 + **자본금 불변** (이익소각) / 미발행주식수 회복 ✕

## C4. 전환종류주식
- 전환권: **주주 + 회사** 모두 가능 / 폐쇄기간 중 전환 가능 (의결권 행사는 경과 후)
- 하향전환: 주식수↑→자본금↑ / 상향전환: 주식수↓→자본금↓
