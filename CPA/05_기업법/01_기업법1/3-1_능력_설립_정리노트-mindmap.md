# Part A. 회사의 능력

## A1. 권리능력

### ★★★ 성질상 제한
- 자연인 ✕ → 친족권/상속권 ✕ / 지배인 ✕ / 발기인 ○
- 유언(유증) 할 수 ✕ / 유언(유증) 받을 수 ○

### 법률상 제한
- 회사가 인적회사의 무한책임사원 = **불가**
- 회사가 LLC의 업무집행사원 = **가능**
- 회사가 물주(유한책임사원·주주) = **가능**

### 목적에 의한 제한
- 판례: 회사 권리능력은 정관 목적범위로 **제한** (제한설)
- 목적수행에 직·간접 필요한 행위까지 포함 인정

## A2. 불법행위능력
- 모든 회사 = 불법행위능력 ○ → 손배책임 가능
- 단, 회사는 일반적인 **범죄능력 ✕** (환경섹터 예외)

### 책임 구조

| 가해자 | 회사 책임 | 개인 책임 |
|--------|---------|---------|
| 대표이사 | §210 연대 | §401 + 민법불법행위 |
| 일반이사 | 사용자책임 | §401 + 민법불법행위 |
| 피용자 | 사용자책임 | 민법불법행위 |

# Part C. 설립무효·취소의 소

## C1. 하자 유형

| 유형 | 소의 종류 | 비고 |
|------|---------|------|
| **객관적 하자** (절/창/하/자) | 설립**무효**의 소 | 주식회사는 객관적 하자만 인정 |
| 주관적 하자 (의사무능력/미성년) | 설립**취소**의 소 | 주식회사 **불가** |
| 사해설립 | 사해설립취소의 소 | 원고: 채권자 / 피고: 회사+사원 |

### 객관적 하자 4가지
- ① 정관 절대적 기재사항 흠결 / ② 모집설립 창립총회 소집 흠결
- ③ 창립총회 결의 하자 / ④ 자본금의 중대한 흠결
- ★★ 물적회사 원시정관 **공증인 인증 흠결**도 해당 (단, 10억 미만 발기설립 → 인증 불필요)

## C2. 소의 특성

| 항목 | 내용 |
|------|------|
| 성질 | **형성소송** (소로만 주장) |
| 피고 | 회사 (사해설립: 회사+사원) |
| 제소기간 | 설립등기일+**2년** |
| 재량기각 | ①하자보완 ②제반사정참작 |

## C3. 판결의 효력

| | 원고승소 | 원고패소 |
|---|--------|--------|
| 효력 | **대세효 + 불소급효** + 청산 | 당사자간만 (악의·중과실 → 연대손배) |

### 원고승소에 소급효 있는 소 (비교)
- 총회결의하자 4종 (취소/무효확인/부존재확인/부당결의취소변경) + 감자무효의 소
