# 📊 CPA 기업법1 — 학습 대시보드

> 이 노트는 **Dataview 플러그인**이 설치되어 있어야 자동 집계됩니다.
> Dataview 설치: 옵시디언 설정 → 커뮤니티 플러그인 → "Dataview" 검색 → 설치 & 활성화

---

## 1. 전체 진행 현황

> 각 정리노트의 frontmatter(`status`, `confidence`, `next_review`)를 기준으로 자동 정렬됩니다.

```dataview
TABLE 
  status AS "상태",
  confidence AS "자신감",
  round AS "회독수",
  next_review AS "다음 복습일",
  length(file.inlinks) AS "연결된 오답수"
FROM "CPA기업법1"
WHERE type = "정리노트"
SORT next_review ASC
```

---

## 2. 오늘 복습해야 할 노트

```dataview
LIST
FROM "CPA기업법1"
WHERE type = "정리노트" AND next_review <= date(today)
SORT next_review ASC
```

---

## 3. 약점 파트 (자신감 '하')

```dataview
TABLE 
  weak_points AS "약점 키워드"
FROM "CPA기업법1"
WHERE type = "정리노트" AND confidence = "하"
```

---

## 4. 미해결 오답 목록

```dataview
TABLE 
  source AS "출처",
  wrong_type AS "틀린 유형",
  related_note AS "관련 노트"
FROM "오답노트"
WHERE resolved = false
SORT date DESC
```

---

## 5. 틀린 유형별 통계

```dataview
TABLE length(rows) AS "개수"
FROM "오답노트"
WHERE resolved = false
GROUP BY wrong_type
```

---

## 6. 최근 백지복원 기록

```dataview
TABLE 
  target_note AS "대상",
  score AS "완성도",
  date AS "날짜"
FROM "CPA기업법1"
WHERE type = "백지복원"
SORT date DESC
LIMIT 10
```

---

## 7. 최근 7일 학습 로그

```dataview
LIST
FROM "일일기록"
WHERE date >= date(today) - dur(7 days)
SORT date DESC
```

---

## 📌 사용법 요약

| 할 일 | 방법 |
|--------|------|
| 공부 시작 | Daily Note 생성 → `daily_log` 템플릿 적용 |
| 문제 틀림 | 새 노트 → `오답노트` 템플릿 → `오답노트/` 폴더에 저장 |
| 백지복원 | 새 노트 → `백지복원` 템플릿 → 기억나는 대로 작성 |
| 회독 완료 | 해당 정리노트 frontmatter에서 `round` +1, `status` 업데이트 |
| 약점 발견 | 정리노트 frontmatter `weak_points`에 추가, `confidence` 조정 |
| 복습일 도래 | 이 대시보드의 "오늘 복습해야 할 노트" 확인 |
