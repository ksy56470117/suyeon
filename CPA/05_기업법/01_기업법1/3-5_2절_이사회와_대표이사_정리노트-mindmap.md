# 3-5. 이사회와 대표이사 — CPA 체계 정리노트


> 관련: [3-4 주주총회](./3-4_1절_주주총회_정리노트.md) · [3-5 감사](./3-5_3절_감사와_감사위원회_정리노트.md) · [3-6 그밖의제도](./3-6_주식회사와_그_밖의_제도_정리노트.md)


# Part A. 이사


## A1. 선임·임기·인원

- 주총결의+동의 / 임기 **3년↑ 불가** / 원칙 3명↑ (10억미만: 1~2명, 각자대표)

## A2. 사외이사

- 상무 종사 ✕ / 결격: 최근2년 종사자·최대주주·로얄패밀리 등
- 상장회사: 1/4↑ / 대규모(2조↑): **과반수+3인↑** / 후보추천위원회 필수

## A3. 종임

- 위임종료/임기만료/사임/회사해산 / **결원(§386)**: 퇴임이사→후임까지 권리의무
- 일시이사: **법원 청구에 의해** (직권 ✕)

## A4. 해임

- **총회특결**(정당이유 불문) / 임기 정한 경우 정당이유 없이 해임 → 손배청구
- **해임판결**: 부정행위/법령정관위반 + 해임결의 부결 → 3/100주주 결의일+1월


# Part B. 이사회


## B1. 소집

| | 주주총회 | 이사회 |
|---|--------|--------|
| 소집권자 | 이사회 | **각 이사** |
| 통지기간 | 2주전 | **1주전** (정관단축가능) |
| 통지방법 | 서면/전자 | **구두 가능** |
| 목적기재 | 필요 | **불필요** |
| 절차생략 | — | 이사+감사 **전원동의** |

## B2. 결의

- 이사 과반수 출석 + 출석이사 과반수 찬성 / 정관으로 비율 높게만 가능
- 특별이해관계인: 의결권 ✕ / 의사정족수 산입 ○ / 의결정족수 **불산입**
- 하자: 상법소 ✕ → **민법 당연무효** (소+소외)

## B3. 위원회

- 정관 정함에 따라 설치(임의) / 대규모: 감사위원회+후보추천위 **필수**
- 감사위원회: **3인↑, 2/3↑ 사외이사** / 결의 → 이사 통지하되 이사회 번복 **불가**
- 위임불가: **총.대.설.정**


# Part C. 대표이사


## C1. 선임·해임

- 원칙: 이사회 / 예외: 정관→주총 / 해임: 선임기관이 해임 (이사회=보통결의)

## C2. 대표권의 성질

- **포괄정형성·불가제한성·영업관련성** → [참조: 2-1 총론-지배인](./2-1_상행위총론_정리노트.md)

## C3. 위법한 대표행위 ★★★★★

### 이사회결의 흠결

- 일반론: **상대적 무효** (내부무효+외부 선악구분) / 증명책임: 회사
- 자기거래 승인 없음: 상대적 무효 / **신주발행**: 무조건 유효
- ★★★★★ **주총결의 흠결**: **깡그리 무효**

### 대표권 남용

- 외형이론(원칙: 회사책임 ○) / 예외: 상대방 악의·(중)과실 → 회사 면책

## C4. 표현대표이사 (§395) ★★★★

- 요건: 대표이사 명칭허락(외관부여) + 이사자격(유추적용으로 비이사도)
- 상대방 범위: 직접+선의 제3취득자 / **소송행위 적용 ✕**
- 등기의 적극적효력과 무관: X가 등기부 미열람 ≠ 중과실

## C5. 공동대표이사

- 수인 선임 + 공동대표 **결의** 필요 (없으면 각자대표)
- 능동대표: 공동 / 수동대표: 단독 가능
- 무권대표: 원칙 회사책임 ✕ / 예외: 등기소극적효력+**추.표.사**

# Part E. 이사의 책임


## E1. 회사에 대한 손배 (§399)

- 고의·**과실**(경과실도) / 결의찬성이사 연대 / 이의미기재 → 찬성**추정**
- 감면: 정관으로 보수 **6배**(사외3배) 초과분 면제 / **법원** 배상액 제한 가능
- 소멸: 총주주동의 면제 / F/S승인후 2년(부정행위 제외) / 시효 10년

## E2. 제3자에 대한 손배 (§401)

- 고의·**중과실** / 증명: 제3자 부담 / 민법 불법행위와 **경합**
- 직접손해주주: 제3자 ○ / 간접손해주주: 제3자 **✕(판례)**

## E3. 업무집행관여자 (§401의2) — "정주영케이스"

- 영향력이용 지시자 / 이사명의 직접집행자 / 명칭사용 집행자 → **이사로 봄**
- 명목상이사도 **연대책임**

## E4. 대표소송 (§403) → 1/100 소수주주권

- 피고: 이사·발기인·통모인수인·감사·청산인·집행임원·업무집행관여자
- 2단계(원칙): 회사청구→30일내 미제기→주주 소제기 / 예외: 즉시 소제기
- 법원허가 없이 소의취하/포기/인낙/화해 불가 / 승소·패소 불문 **회사에 효력**

## E5. 집행임원제도 (EO)

- 이사회결의로 선임 / 이사겸임 가능(사외이사 ✕) / CEO=대표권
- 임기 **2년↑불가** / 보수: 이사회 결정 / 등기사항
