# 📘 옵시디언 셋업 가이드 — CPA 기업법1

## 1단계: Vault에 폴더 배치

다운로드한 파일들을 아래 구조로 옵시디언 Vault에 넣으세요.

```
내 Vault/
├── 00_대시보드.md              ← 메인 현황판
├── 플래시카드_두문자키워드.md   ← SR 플러그인용 카드
│
├── CPA기업법1/                 ← 정리노트 16개 (frontmatter 포함)
│   ├── 2-1_상행위총론_정리노트.md
│   ├── 2-2_상행위각론_정리노트.md
│   ├── 3-1_회사법_통칙_정리노트.md
│   ├── ...
│   └── 3-6_주식회사와_그_밖의_제도_정리노트.md
│
├── 오답노트/                   ← 틀린 문제 기록 (직접 생성)
│
├── 일일기록/                   ← Daily Note 학습 로그 (직접 생성)
│
└── templates/                  ← 템플릿 3종
    ├── daily_log.md
    ├── 오답노트.md
    └── 백지복원.md
```

`오답노트/`, `일일기록/` 폴더는 비어있는 상태로 직접 만들어주세요.

---

## 2단계: 필수 플러그인 설치

옵시디언 설정 → 커뮤니티 플러그인 → 제한 모드 해제 → 검색 & 설치

| 플러그인 | 용도 | 우선순위 |
|----------|------|----------|
| **Dataview** | 대시보드 자동 집계, 복습일 관리 | ⭐ 필수 |
| **Templater** | 템플릿에서 날짜 자동 삽입 | ⭐ 필수 |
| **Spaced Repetition** | 두문자 키워드 플래시카드 | ⭐ 강추 |
| **Calendar** | 사이드바에 달력, Daily Note 연동 | 선택 |
| **Kanban** | 학습 진행 보드 (미학습→진행→완료) | 선택 |

---

## 3단계: 플러그인 설정

### Templater 설정
1. 설정 → Templater → Template folder location: `templates`
2. Trigger Templater on new file creation: ON

### Dataview 설정
1. 설정 → Dataview → Enable JavaScript Queries: ON
2. Enable Inline Queries: ON

### Spaced Repetition 설정
1. 설정 → Spaced Repetition → Flashcard tags: `#card`
2. 구분자: `::`

---

## 4단계: 일일 학습 루틴

### 매일 시작할 때
1. `Ctrl/Cmd + P` → "Templater: Create new note from template" → `daily_log` 선택
2. 파일 이름: `2026-04-03` (날짜) → `일일기록/` 폴더에 저장
3. "오늘 학습 범위"에 계획 기록

### 공부하면서
4. 정리노트를 읽고, 키워드를 소리 내어 암기
5. 한 파트 끝나면 → 새 노트 → `백지복원` 템플릿 → 기억나는 대로 작성
6. 원본과 대조 후 빠진 부분 기록

### 문제 풀고 나서
7. 틀린 문제마다 → 새 노트 → `오답노트` 템플릿 → `오답노트/` 폴더에 저장
8. `related_note` frontmatter에 정리노트 링크 추가
9. `wrong_type`에 `지식부족` / `유사개념혼동` / `선지함정` 중 하나 기록

### 하루 마무리
10. Daily Note에 백지복원 결과, 오답 수확, 내일 할 것 기록
11. 공부한 정리노트의 frontmatter 업데이트:
    - `round`: +1
    - `status`: 1회독완료 / 2회독완료 / 백지복원완료
    - `confidence`: 상 / 중 / 하
    - `weak_points`: 약한 키워드 추가
    - `next_review`: 다음 복습일 (간격: 1일→3일→7일→14일)

### 다음 날 시작할 때
12. `00_대시보드.md` 열어서 "오늘 복습해야 할 노트" 확인
13. 미해결 오답 목록 확인 → 복습 완료되면 `resolved: true`로 변경

---

## 5단계: 플래시카드 복습

1. `플래시카드_두문자키워드.md` 열기
2. 왼쪽 리본의 카드 아이콘 클릭 (Spaced Repetition)
3. 카드가 나오면 답을 먼저 떠올린 후 확인
4. 난이도 선택: Easy / Good / Hard → 자동으로 복습 간격 조절
5. 새 키워드를 발견하면 같은 형식으로 카드 추가:
   ```
   질문 #card
   :: 답변
   ```

## 팁

- **백링크 활용**: 정리노트를 열고 오른쪽 사이드바의 "Backlinks" 패널을 보면, 그 노트를 참조하는 오답노트/백지복원 기록이 한눈에 보입니다. 백링크가 많은 섹션 = 약점입니다.
- **그래프 뷰**: `Ctrl/Cmd + G`로 전체 노트 연결 관계를 시각화할 수 있습니다. 오답노트가 특정 정리노트에 밀집되어 있으면 해당 파트를 집중 공략하세요.
- **태그 검색**: `#weak` 태그로 약점 백지복원 기록만, `#지식부족` 등으로 오답 유형별 필터링이 가능합니다.
- **시험 직전**: 대시보드에서 `confidence: 하`인 노트 + 미해결 오답만 집중 복습하면 됩니다.
