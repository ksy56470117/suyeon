# Part A. 3요소 — 자본금·주식·주주유한책임

## A1. 자본금
- 규범적/당위적 개념 (cf. 순자산=사실적 개념)
- 액면주: 자본금 = 액면가 × 발행주식수
- 무액면주: 발행가액 중 자본금 항목 계상금액 (설립시=발기인전원동의 / 신주발행=이사회)
- 자본금 = 등기사항 ○ / **정관 절대적 기재사항 ✕**

## A2. 수권자본제도 (§416)
- 주총이 이사회에 신주발행권한을 max까지 수여
- 폐지: 설립시 1/4↑ 발행 / 발행예정주식수 변경시 4배 한도 → 현행법 폐지

## A3. 자본금의 3원칙

| 원칙 | 내용 |
|------|------|
| 자본확정 | 한국 해당 ✕ (자본금 = 정관 절대적 기재사항 ✕) |
| **자본유지(충실)** | 가장 핵심적 이념 |
| 자본불변 | 증자: 이사회(비엄격) / **감자: 총회특결+채보절차(엄격)** |

## A4. 주주유한책임
- 인수가액 한도 책임 = **강행규정** / 위반 정관·총회결의 = 무효
- 자발적 채무분담은 가능

# Part B. 설립방법·절차

## B1. 발기설립 vs 모집설립

| | 발기설립 | 모집설립 |
|---|--------|--------|
| 청약방식 | 서면 | 주식청약서 |
| 납입해태 | 강제이행 | 실권절차(§307) |

## B2. 정관의 작성

### ★★★★ 절대적 기재사항 (목/상/예/일/본/공 + 역사적 사실 2개)

| 정관기재 ○ + 등기 ○ | 목적/상호/발행예정주식총수/1주금액(액면주)/본점소재지/공고방법 |
|---|---|
| 정관기재 ○ + 등기 ✕ | 발기인 성명·주소(역사적) / 설립시 발행주식총수(역사적) |

### 물적회사 원시정관 — 공증인 인증 필요
- 예외: 자본금 10억 미만 **발기설립** / 자본금 10억 미만 유한회사 → 인증 불필요

## B3. 주금납입

| 항목 | 내용 |
|------|------|
| 현실납입 | 주주전원동의로도 면제 ✕ / 대물변제 ✕ (자본충실) |
| 전액납입 | 분할납입 금지 |
| 반대채권 상계 | 설립시: 특별절차 없이 ○ / 신주발행시: 회사동의 필요 |
| 납입해태 | 발기=강제이행 / 모집=실권절차 |

## B4. 설립등기 — **창설적+보완적+해제적 효력**

# Part C. 설립중의 회사

## C1. 개념
- 강학상 개념(조문 ✕) / 판례: 권리능력 없는 사단
- 형성시기(판례): 발기인이 **1주 이상 인수**한 때 ("1주설")

## C2. 발기인의 대표권 범위 ★★★★
- 판례 = **광의설**: 개업준비행위(성립후 회사를 위한 행위)까지 포함

## C3. 권리의무의 승계 (동일성설)
- 승계요건: ①설립중회사 존재(1주설) ②대표권 범위 내 행위
