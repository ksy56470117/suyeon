# Part A. 변태설립사항

## A1. 종류·규제
- 종류: 현물출자 / 재산인수 / 설립비용 / 발기인의 특별이익·보수
- 규제: **"변태설립은 정/검/청"** — ①정관기재(상대적) ②검사인조사 ③주식청약서기재

## A2. 현물출자
- 과거 발기인만 가능 → 현행법 모집주주도 가능
- 정관기재: ★**출자자 성명** + 목적재산 종류·가격 + 부여할 주식 종류·수
- 과대평가: 등기 전 → 법원/창립총회 변경 / 등기 후 → 발기인·이사·감사·검사인 연대책임
- 출자이행: 납입기일 지체없이 재산인도 + 서류완비·교부 (이전등기는 **회사가 셀프**)

## A3. 재산인수
- 발기인이 "회사성립 후 양수하겠다"는 계약(개인법적 행위=매매)
- 정관기재: ★**양도인의 성명** + 재산종류·수량·가격
- 판례: 정관 미기재 → 무효 / 사후설립에도 해당 → **주총특결로 추인 가능**

## A4. 설립비용·특별이익·보수
- 설립비용: 개업준비비용 ✕ / 판례(회사전액부담설): 초과분도 회사→채권자 지급 후 발기인에 구상
- 특별이익: 설립등기 후 지급 / 보수: 설립등기 무관

# Part B. 가장납입

## B1. 통모가장납입
- 발기인+은행 공모 → 납입금보관자 책임(§318): 대항 불가

## B2. 일시차입금 가장납입(위장납입) ★★★

| 논점 | 납입유효설(판례) | 납입무효설(통설) |
|------|-------------|-------------|
| 납입 효력 | 유효(완료) | 무효(미완료) |
| 설립무효 | 해당 ✕ | 해당 가능 |
| 자본충실책임 | 부담 ✕ | 부담 ○ |

### 가장납입 관계자 책임
- **손배책임(§322)**: 회사에 경과실도 / 제3자에 악의·중과실
- 체당납입주금 상환의무 ○
- 형사: 특별배임죄 ○ / 납입가장죄 ○ / 횡령죄 ✕ / 업무상횡령 ✕

# Part C. 발기인의 책임 → [참조: 3-5 이사책임](./3-5_2절_이사회와_대표이사_정리노트.md)

## C1. 자본충실책임 (§321) — **무과실책임**

| | 상황 | 결과 |
|---|------|------|
| **인수담보** | 인수 깨짐(소주만병/만16세) | 발기인이 "인수인" → **발기인이 주주** |
| **납입담보** | 납입 안 됨(가난뱅이) | 발기인 연대 납입 → **가난뱅이(원래 인수인)가 주주** |

> ★ 인수인 이름표 → 주주 이름표

## C2. 손배책임 (§322) — **과실책임**

| 대상 | 요건 |
|------|------|
| to 회사 | 임무해태 (**경과실** 포함) |
| to 제3자 | **악의 or 중과실** |

## C3. 불성립시 책임 (§326) — 무과실책임
- 설립 관련 행위 연대책임 / 지급비용 발기인 부담

## C4. 유사발기인 (§327)
- 자본충실책임 ○(연대) / 손배책임 ✕(과실 없으므로) / 불성립시 ○(연대)
