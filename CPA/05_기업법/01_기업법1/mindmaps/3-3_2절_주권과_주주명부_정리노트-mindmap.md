# 3-3. 주권과 주주명부 — CPA 체계 정리노트


> 관련: [3-3 주식개관](./3-3_1절_주식과_주주의_개관_정리노트.md) · [3-3 양도·제한](./3-3_3절_주식의_양도와_제한_정리노트.md)


# Part A. 주권


## A1. 주권의 성질

- 비제시증권·비상환증권·비문언증권·**비설권증권**
- ★ 주권 발행 ≠ 주주권 발생 조건

## A2. 효력발생시기 비교

| | 시기 |
|---|------|
| 주식(right) — 설립시 | 설립등기 한 때 |
| 주식(right) — 신주발행시 | 납입기일의 **다음날** |
| 주권(paper) | **진정한 주주에게 교부**된 때 |

## A3. 주권의 선의취득

- 유효한 주권 + 교부 + 양도인이 무권리자/무권대리인 + 양수인 선의·무중과실
- 선의취득 대상 ✕: 권리주단계 발행/분실·도난 과정/위조/제권판결 실효/불소지신고

## A4. 제권판결

- 소극적효력: 주권 → 쓰레기 / 적극적효력: 판결문 소지 = 주권소지
- 판례: **제권판결취득자 우선설** / 통설: 선의취득자 우선설

## A5. 불소지제도

- 정관에 다른 정함 없으면 가능 / 신고: 명의개서 필한 주주 (인수인도 가능)
- 불소지신고한 주권이 잘못 유통 → ✕ 선의취득 ✕ 제권판결

## A6. 전자등록 (§356-2)

- 정관근거 필요 / 양도등록·입질등록 → 효력발생 / 추정력·선의취득 인정


# Part B. 주주명부·명의개서


## B1. 명의개서

- 청구: **양수인 단독** (양도인 ✕) / 주권발행 ○ → 주권제시 / 주권발행 ✕ → 양도증명

## B2. 명의개서의 효력

| 효력 | 내용 |
|------|------|
| 대항력 (§337) | 명의개서 ✕ → 회사에 대항 못함 |
| 추정력 | 주주명부 기재 → 적법한 주주 추정 |
| **면책력 (§353)** | 주주명부 기준 통지·배당 → 회사 면책 |
| 창설적효력 ✕ | 허위 명의개서 → 주주권 발생 ✕ |

## B3. 주주명부 폐쇄와 기준일 (§354)

- 폐쇄기간: **3개월** 초과 불가 (초과분만 무효)
- 기준일: 권리행사일 앞선 3개월 내의 날
- 폐쇄 중: 명의개서·등록질 ✕ / 주소변경·불소지신고 ○

## B4. 명의개서 미필주주

- 원칙: 주주명부 기준 / 예외(**부당거절**) → 미필주주도 권리행사 인정
- 이익귀속: **양수인귀속설(통설)** → 반환청구 가능
