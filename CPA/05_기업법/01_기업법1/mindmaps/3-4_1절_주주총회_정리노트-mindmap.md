# 3-4. 주주총회 — CPA 체계 정리노트


> 관련: [3-5 이사회](./3-5_2절_이사회와_대표이사_정리노트.md) · [3-5 감사](./3-5_3절_감사와_감사위원회_정리노트.md) · [3-1 구조조정](./3-1_구조조정_정리노트.md)


# Part A. 총회의 권한·소집


## A1. 총회 vs 이사회 권한

- 총회: 법률/정관이 정하는 사항에 한정 / 이사회: **주된 결정기관**
- 총회→이사회 위임: **불가** / 이사회→총회: **가능** (정관에 의해)

## A2. 소집결정 — 원칙 이사회 / 예외 **소.감.법**

| 소집권자 | 절차 |
|---------|------|
| 소수주주(3/100) | 이사회 청구 → 법원허가 소집 |
| 감사 | 이사회 청구 → 법원허가 소집 |
| 법원명령 | 직접 |

## A3. 소집통지

- 총회일 **2주 전** 서면/전자문서 / 10억미만: **10일 전**
- 10억미만 특례: 주주전원동의 → 절차생략 / 서면결의 가능
- 상장회사: 1/100↓ 주주 → **공고로 대체** 가능


# Part B. 주주제안권·의결권


## B1. 주주제안권 (소수주주권 3/100)

- 총회일 **6주 전** 서면/전자문서 / 의안 설명기회 보장
- 이사회 거부 가능: 한.삼.고.소.해

## B2. 의결권 제한 — ★★★★★ **종.자.특.감**

## B3. 의결권 행사방법

| 방법 | 요건 |
|------|------|
| 대리행사 | 위임장(원본) 제출 |
| 서면행사(투표) | 정관규정 |
| 서면결의 | 전원동의 (10억미만만) |
| 전자행사(투표) | 이사회결의 |
| 불통일행사 | 3일전 통지 |


# Part C. 결의 정족수 ★★★

| 결의 | 발행주식총수 | 출석의결권수 |
|------|---------|---------|
| **보통결의** | 1/4↑ | 과반수 |
| **특별결의(종류주총)** | 1/3↑ | 2/3↑ |
| **특수결의**(책임면제/조직변경) | 100% | — |
| 이사회 | 과반수 **출석** | 과반수 찬성 |
| 창립총회 | 인수주식총수 과반수 | 출석 2/3↑ |


# Part D. 영업양도 특별결의·주매청


## D1. 영업양도 등 → 총회특결 (강행법규)

- 영업 전부/중요일부 양도 / 영업임대·경영위임·손익공통계약 / 영업양수
- 회사존속기초재산 처분도 총회특결 (담보제공은 원칙 불필요, 매도담보 예외)

## D2. 주매청 — **구.영.승** → [참조: 3-1 구조조정](./3-1_구조조정_정리노트.md)

- 구조조정 반대주주 / 영업양도등 반대주주 / 주식양도 승인거부
- 총회결의일+20일 내 / 회사 매수: 매수청구기간종료일+2월 내
- ★★★ **악법3종set**: 소규모(합병/분할합병/주식교환) → 주매청 ✕


# Part E. 종류주주총회 — 정.구.회


## E1. 결의 요하는 상황 (8가지)

- 정관변경 / 구조조정(4) / 회덮밥조문(3: 신주인수·병합분할소각·합병분할배정)

## E2. 결의 흠결

- 일반총회+종류총회 = **각각 독립 요건** → 종류총회 흠결 → 정관변경효력 ✕ / 일반총회는 유효


# Part F. 총회결의의 하자 ★★★

| 하자 | 절차(경미) | 절차(중대) | 내용(정관위반) | 내용(법령위반) |
|------|---------|---------|----------|----------|
| 소 | 결의**취소**의 소 | **부존재**확인의 소 | 결의**취소**의 소 | **무효**확인의 소 |

### 결의취소의 소

- 원고: 주주/이사/감사 / 제소: 결의일+**2월** / **재량기각 가능**(제반사정만으로)
- 판결: 대세효 + **소급효** + 등기(본점)

### 결의무효·부존재확인의 소

- **확인소송** → 소+소외 가능 / 제소기간 **무제한** / 재량기각 **불가**

### 부당결의취소변경의 소

- 특별이해관계주주 의결권 ✕ → 현저히 부당한 결의 / 2월 / **재량기각 불가**
