# 2-1. 상행위총론 — CPA 체계 정리노트


> 관련: [2-2 상행위각론](./2-2_상행위각론_정리노트.md) · [3-1 통칙](./3-1_회사법_통칙_정리노트.md)


# Part A. 상행위의 개념


## A1. 의의

- 영리의 목적(돈벌려고)으로 행해지는 **법률행위**
- 영업으로 하는 상호부금(곗돈모으기) = 기본적 상행위

## A2. 주관주의

- 상인을 먼저 정의 → 상행위 개념 정함 = 주어가 **"상인"**

| 구분      | 내용                    |
| ------- | --------------------- |
| 영업적 상행위 | 상인이 **영업으로** 하는 행위    |
| 보조적 상행위 | 상인이 **영업을 위하여** 하는 행위 |


### ★★★★★ 핵심

- 상인이 영업을 위하여 하는 행위 → 상행위로 **본다(간주)**
- 상인의 행위 → 영업을 위하여 하는 것으로 **추정** (간주 ✕)

## A3. 일방적 vs 쌍방적 상행위


| 구분 | 내용 | 예시 |
| ------ | --------------------- | -------------- |
| 일방적 | 1인에게만 상행위 → 전원에 상법 적용 | 상사법정이율 |
| 쌍방적 | 둘 다 상인일 때만 적용 | 일반상사유치권(카센터사건)|


# Part B. 상행위의 특칙


## B1. 민법 특칙 유무


| 특칙 ○ | 특칙 ✕ |
|--------|--------|
| 유질계약허용, 비현명주의, 무상수치인의무, 법정이율 | **기간계산의 방법** |


## B2. 적용범위별 규정


| 쌍방적에만 적용 | 일방적에도 적용 |
|-------------|------------|
| 일반상사유치권 피담보채권 | 상사법정이율 |
| 확정기매매 해제 | 대여금/체당금 법정이자 |
| 상사매매 매도인 공탁/경매권 | 상인의 보수청구권 |


## B3. 상사소멸시효 (§64)

- 상행위로 인한 채권 → 다른 규정 없으면 **5년** (민사 10년)
- 범위: 가장 상행위성 옅은 (일방적+보조적)에도 적용

### 상사시효 5년 적용 ○

- ① 채무불이행 → 손배청구권
- ② 계약해제 → 원상회복청구권
- ③ 급부 자체의 부당이득반환청구권

### 적용 ✕ (Not 상행위채권)

- 불법행위 → 손배청구권
- 상행위 무관 부당이득 반환청구권

## B4. 상사법정이율 (§54)

- 상행위로 인한 채무 법정이율 = **연 6%**
- 일방적+보조적 상행위에도 적용

## B5. 상사유치권 (§58) ★★


### 성립요건 (**당.피.목.관.반**)


| 요건    | 내용                         |
| ----- | -------------------------- |
| **당** | 채권자·채무자 **모두 상인**          |
| **피** | **쌍방적** 상행위로 발생            |
| **목** | 채무자 소유 + 채권자 **상행위로** 점유취득 |
| **관** | 견련성 **불필요**                |
| **반** | 반대특약 없을 것                  |

- ※ 선행저당권자 >>> 유치권자
- 일반상사유치권: 모든 상인 / 특별상사유치권: 대리상·위탁매매인·운송인·운송주선인 → [2-2 각론](./2-2_상행위각론_정리노트.md)

## B6. 상사질권 (§59)

- **유질계약 유효** (민법: 무효)
- 질권설정자 **상인 아닌 경우에도** 유효
- 최소 일방적 상행위만 있으면 유질계약 가능

## B7. 상사채무 연대책임 (§57)

- **채무자에게** 상행위가 될 때만 적용 (채권자 무관)
- 조합채무가 조합원 전원 위해 상행위 → 조합원 연대책임 ○
- 사례: **짜장면사건(온정주의)** — 학생들은 상인 ✕ → 연대책임 없음

## B8. 상사보증인 연대책임 (§57)

- ① 보증이 상행위이거나 ② 주채무가 상행위 → 연대 변제 책임
- 민법: 최고/검색의 항변 ○ / 상법: **항변 ✕** (순서불문+비율불문)
- 사례: **신원보증사건(삼촌&조카)**

## B9. 보수청구권 (§61)

- 상인이 ① 영업범위 내에서 ② 타인을 위하여 행위 → 상당한 보수 청구

## B10. 그 밖의 특칙


| 항목               | 핵심                              |
| ---------------- | ------------------------------- |
| **대리방식(§48)**    | 비현명주의 (단, 어음행위=엄격한 현명주의)        |
| **위임(§49)**      | 본지에 반하지 않으면 위임받지 않은 행위 가능       |
| **대리권존속(§50)**   | 본인 사망으로 소멸 ✕                    |
| **낙부통지의무(§53)**  | 상시거래 + 영업부류 청약 → 지체없이 / 해태=승낙간주 |
| **물건보관의무(§60)**  | 청약 거절해도 청약자 비용으로 보관             |
| **법정이자청구권(§55)** | 금전대여→법정이자 / 금전체당→체당일 이후         |
| **임치 주의의무(§62)** | 영업범위 내 무상임치도 **선관주의**           |

# Part C. 상사매매


## C1. 의의·요건

- 상인간 상행위로서의 매매 / 취지: **신속종결+매도인보호**
- 요건: ① 쌍방 상인 ② 쌍방적 상행위 매매 ③ 반대특약 없을 것 (임의규정)

## C2. 매도인의 공탁/경매권 (§67)


| 구분 | 상법 | 민법 |
|------|------|------|
| 경매 보충성 | ✕ (바로 가능) | ○ (공탁 우선) |
| 법원허가 | 불필요 | **필수** |
| 매매대금 충당 | 가능 | 전부공탁 |


## C3. 확정기매매 해제 (§68)


| | 최고 | 기간경과 | 해제의사표시 |
|---|:---:|:---:|:---:|
| **상법** | ✕ | ✕ | ✕ |
| 민법-일반 | ○ | ○ | ○ |
| 민법-정기행위 | ✕ | ✕ | ○ |


## C4. 매수인 검사/통지의무 (§69) ★

- 수령 시 지체없이 검사 → 하자/수량부족 즉시 통지 (즉시 발견불가 → 6월 내)
- **간접의무**: 불이행해도 강제이행/손배 ✕ (단, 해제·감액·손배 청구 불가)
- 부대체물 적용 ✕ / 매도인 악의 시 적용 ✕

## C5. 매수인 보관/공탁/경매의무 (§70,71)

- **진정한 의무**: 위반 시 손배 책임
- §70: 계약해제 → 매도인 비용 보관/공탁 (멸실·훼손 → 법원허가 경매)
- §71: 목적물 상위/수량초과 → 보관/공탁/경매

# Part D. 특수한 계약


## D1. 상호계산 (§72)

- 상인간 or 상인+비상인 / 상시거래 / 채권채무 상계 → 잔액 지급

### 불가분의 원칙

- 독립성 상실: 채권행사 ✕ / 임의제거 ✕ / 시효진행 ✕ (잔액확정 때부터)
- 어음/수표채권 = 박스에 넣을 수 없음 / 어음으로 "인한" 채권 = 독립성 ○

### 기간·해지

- 약정기간 or 6개월 / 해지: **언제든지** (기간 정했어도 예고없이)
- 잔액확정: 원칙 이의제기 불가 / 예외 착오/탈루 시 이의 가능

## D2. 익명조합 (§78) → [참조: 3-1 통칙-합자회사](./3-1_회사법_통칙_정리노트.md)


### 핵심


| 항목 | 내용 |
|------|------|
| 이익분배 | **필수** (손실 분담은 필수 ✕) |
| 출자 | 노무/신용 **불가** |
| 출자재산 | 영업자 **단독소유** |
| 영업거래 | 영업자 단독거래 (익명조합원 원칙 책임 ✕) |
| 경업금지 | **명문규정 없음** |


### 이익배당/손실 (§82) — 임의규정

- ① 출자 손실 감소 → 전보 후 아니면 이익배당 ✕
- ② 손실>출자액 → 이미 받은 이익 반환/증자 의무 ✕

### 해지 (§83) / 종료 (§84)

- 기간미정/종신 → 영업연도 말 (6월전 예고) / 부득이 → 언제든지
- 종료: 영업폐지/양도, 익명조합원 파산, 영업자 **사·금·파**
- 영업자: 양도 ✕ 상속 ✕ / 익명조합원: 양도 ✕ **상속 ○**

### 준용 (§86) — 합자회사 유한책임사원

- 출자: 노무/신용 ✕ / 대표행위: 집행권 ✕(임의) 대표권 ✕(**강행**)
- 감시권: 원칙 영업연도 말 / 예외 중요사유 → 법원허가 언제든지

# Part E. 비교표


## E1. 연 6% 법정이자


| 발생 ○ | 발생 ✕ |
|--------|--------|
| 상행위 채권 | 불법/상행위무관 채권 |
| 대여금/체당금 + 채권자=상인 | 대여금/체당금 + 채권자≠상인 |


## E2. 해지 시기 비교


| 계약 | 시기 | 예고 |
|------|------|------|
| 익명조합 | 영업연도 말 | 6월 전 |
| 대리상 | ✕ | 2월 전 |
| 상호계산 | 언제든지 | 불필요 |
